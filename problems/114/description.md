## 문제 설명

공백으로 구분된 문자열 `문장`을 **단어 단위로** 뒤집어 말하기로 출력하세요. 각 단어의 글자 순서는 유지, 단어들의 순서만 역순으로 배치합니다.

- 단어 사이 공백은 정확히 한 칸
- 앞·뒤 공백은 없다고 가정 (또는 연속 공백이 없음)

### 입력
- `문장`: 공백으로 구분된 단어 하나 이상의 문자열 (채점 시 자동 설정)

### 출력
- 단어 순서만 역순으로 뒤집은 문자열 말하기 (공백 1칸으로 연결)

### 예시
`"안녕 엔트리 반가워"` → 출력: `반가워 엔트리 안녕`
`"hello world"` → 출력: `world hello`
`"one"` → 출력: `one`

### 힌트
- 단어들을 순서대로 보조 **리스트**에 모은 뒤, 마지막부터 첫 번째로 되짚으며 결과 문자열에 이어붙이기
- 단어들 리스트·현재 단어·결과 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 단어 리스트를 빈 리스트로, 현재 단어를 `""` 로 초기화
- 문장을 한 글자씩 순회:
  - 공백 만나면 현재 단어를 리스트에 추가하고 현재 단어 초기화
  - 아니면 현재 단어에 글자 이어 붙이기
- 반복 종료 후 마지막 단어도 리스트에 추가
- 결과 문자열을 `""` 로 시작해, 리스트를 **뒤에서 앞으로** 순회하며 단어를 붙이기 (두 번째 단어부터는 앞에 공백 한 칸)

</details>

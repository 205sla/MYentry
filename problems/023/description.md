## 문제 설명

`숫자들` 리스트에서 **연속된 부분 구간**의 합이 가장 큰 값을 구해 말하기로 출력하세요. 이를 **최대 부분합** 문제라고 부릅니다.

- "연속된 부분"이란 리스트에서 이어붙은 항목들의 한 구간입니다.
  - 예: `[1, 2, 3, 4]`의 연속 부분 구간 예시 → `[1, 2]`, `[2, 3, 4]`, `[3]` 등
  - 반례(연속 아님): `[1, 3]`은 건너뛰었으므로 허용되지 않습니다
- 빈 구간은 선택할 수 없습니다 — 최소한 한 개의 항목은 반드시 포함해야 합니다

### 입력
- `숫자들`: 1개 이상의 정수 (음수 포함, 채점 시 자동 설정)

### 출력
- 최대 부분합을 말하기

### 예시
입력: `[1, 2, 3]` → `6` (전체 구간 `[1,2,3]`의 합이 최대)
입력: `[-2, 1, -3, 4, -1, 2, 1, -5, 4]` → `6` (부분 구간 `[4,-1,2,1]`의 합)
입력: `[-1, -2, -3]` → `-1` (모두 음수라면 가장 덜 작은 한 항목)

### 힌트
- 리스트 `숫자들`은 **기본으로 제공**됩니다 (채점 시 값이 교체됨)
- 풀이에 필요한 변수 `현재합`, `최댓값`, `현위치`도 미리 만들어 제공됩니다
- 핵심 아이디어: 리스트를 **한 번만 순회**하면서 두 값을 유지합니다
  - `현재합`: "이번 항목까지 이어 붙였을 때의 최선의 부분합"
  - `최댓값`: "지금까지 본 `현재합` 중 가장 큰 값"
- 매 항목마다 선택: `현재합`을 **이번 항목 하나로 새로 시작**할지(`현재합 = 숫자들 항목`), 아니면 **이전 `현재합`에 이어붙일지**(`현재합 = 현재합 + 숫자들 항목`) — **둘 중 큰 쪽**을 고릅니다
- 그 다음 `최댓값`과 `현재합`을 비교해 더 큰 쪽을 `최댓값`에 저장
- **초기값 주의**: `현재합`과 `최댓값`을 **모두 `숫자들`의 1번째 항목**으로 시작해야 합니다 — `0`으로 두면 모든 항목이 음수인 케이스에서 답이 잘못됩니다
- `흐름` 카테고리의 `N번 반복하기`, `판단` 카테고리의 `>` 블록을 활용하세요

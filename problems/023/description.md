## 문제 설명

`숫자들` 리스트에서 **연속된 부분 구간**의 합이 가장 큰 값을 구해 말하기로 출력하세요. 이를 **최대 부분합** 문제라고 부릅니다.

- "연속된 부분"이란 리스트에서 이어붙은 항목들의 한 구간입니다.
  - 예: `[1, 2, 3, 4]`의 연속 부분 구간 → `[1, 2]`, `[2, 3, 4]`, `[3]` 등
  - 반례(연속 아님): `[1, 3]`은 건너뛰었으므로 허용되지 않습니다
- 빈 구간은 선택할 수 없습니다 — 최소 한 개 항목은 반드시 포함

### 입력
- `숫자들`: 1개 이상의 정수 (음수 포함, 채점 시 자동 설정)

### 출력
- 최대 부분합을 말하기

### 예시
입력: `[1, 2, 3]` → `6`
입력: `[-2, 1, -3, 4, -1, 2, 1, -5, 4]` → `6` (구간 `[4,-1,2,1]`)
입력: `[-1, -2, -3]` → `-1` (덜 작은 한 항목)

### 힌트
- 리스트를 한 번만 순회하면서 "지금까지의 최선"을 두 값으로 갱신하는 방법
- 누적값·최댓값을 담을 보조 변수는 직접 만들어 쓰세요

- 두 변수 유지:
  - 누적합: "이번 항목까지 이어붙였을 때의 최선의 부분합"
  - 최댓값: "지금까지 본 누적합 중 가장 큰 값"
- 매 항목마다 선택:
  - **이번 항목 하나로 새로 시작** vs **이전 누적합에 이어붙이기**
  - 둘 중 큰 쪽을 누적합으로
- 그 후 누적합이 최댓값보다 크면 최댓값 갱신
- **초기값 주의**: 누적합·최댓값을 모두 `숫자들의 1번째 항목`으로 시작. `0`으로 두면 모든 항목이 음수일 때 답이 틀립니다

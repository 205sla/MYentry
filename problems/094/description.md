## 문제 설명

리스트 `숫자들`에서 **서로 다른 값의 개수**(고유 값 개수)를 세어 말하기로 출력하세요.

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 고유 값 개수를 말하기

### 예시
`[1, 1, 2, 2, 3]` → 출력: `3`
`[5, 5, 5, 5]` → 출력: `1`
`[1, 2, 3, 4, 5]` → 출력: `5`

### 힌트
- 이중 반복: 각 위치의 값이 **이전에 등장한 적 없는** 값이라면 카운트 +1
- 카운터·중복 플래그 변수는 직접 만들어 쓰세요
- `판단`의 `=` 블록

<details>
<summary>자세한 접근법</summary>

- 카운터를 `0`
- 바깥 반복 `i = 1..숫자들 항목 수`:
  - 중복 플래그를 `0`
  - 안쪽 반복 `j = 1..(i-1)`: `숫자들의 j번째 = 숫자들의 i번째` 이면 중복 플래그 `1`
  - 중복 플래그 = `0` 이면 카운터 +1 (처음 등장한 값)
- 반복 끝난 뒤 카운터 말하기

</details>

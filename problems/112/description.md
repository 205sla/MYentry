## 문제 설명

공백으로 구분된 **후위표기식** 문자열 `식`을 계산해 결과를 말하기로 출력하세요.

- 숫자와 연산자는 공백 1칸으로 구분
- 연산자는 `+`, `-`, `*` 세 가지
- 토큰은 한 글자 이상 (정수는 여러 자릿수 가능, 모두 양의 정수)
- 올바른 후위표기식이라고 보장됨 (중간에 토큰이 오류 없음)

### 입력
- `식`: 후위표기식 문자열 (채점 시 자동 설정)

### 출력
- 계산 결과를 말하기

### 예시
`"3 4 +"` → 출력: `7`
`"3 4 + 2 *"` → 출력: `14` ( (3+4)×2 )
`"5 1 2 + 4 * + 3 -"` → 출력: `14` ( 5 + (1+2)×4 - 3 )

### 힌트
- 1차원 리스트를 **스택**처럼 사용
- 토큰을 공백 기준으로 하나씩 떼어내며 처리
  - 숫자 → 스택에 push
  - 연산자 → 스택에서 두 개 pop, 계산 후 결과를 push
- 주의: `A B -` 는 `A - B` (스택에서 먼저 꺼낸 게 오른쪽 피연산자)

<details>
<summary>자세한 접근법</summary>

- 스택(리스트)과 현재 토큰(문자열) 변수를 준비
- 식을 앞에서부터 훑으며:
  - 공백을 만나면 현재 토큰을 처리:
    - `+`·`-`·`*` 이면 스택에서 두 값 꺼내 계산 후 push
    - 그 외는 숫자 문자열이므로 수로 변환해 push
  - 공백이 아니면 현재 토큰에 글자 이어붙이기
- 식 끝에 마지막 토큰도 같은 방식으로 처리
- 스택에 남은 하나의 값이 결과

## 문제 설명

과목별 학점이 문자열로 담긴 리스트 `학점들`을 점수로 바꿔 **총합**을 말하기로 출력하세요.

- `A` → `4`
- `B` → `3`
- `C` → `2`
- `D` → `1`
- `F` → `0`

### 입력
- `학점들`: `"A"`·`"B"`·`"C"`·`"D"`·`"F"` 중 하나로 이루어진 리스트 (채점 시 자동 설정)

### 출력
- 학점 점수 총합을 말하기

### 예시
`["A", "B", "C"]` → 출력: `9` (4+3+2)
`["F", "F", "F"]` → 출력: `0`
`["A"]` → 출력: `4`
`[]` → 출력: `0`

### 힌트
- 리스트를 순회하며 각 학점을 해당 점수로 변환해 누적
- `흐름`의 `만약~이라면` 블록 중첩 또는 연속된 `아니면 만약` 사용
- 합 변수는 직접 만들어 쓰세요

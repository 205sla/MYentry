## 문제 설명

리스트 `숫자들`의 각 수에 대해 **모든 약수의 합**(자기 자신 포함)을 계산해 같은 순서로 리스트 `합들`에 담으세요.

### 입력
- `숫자들`: 1 이상의 자연수 리스트 (채점 시 자동 설정)
- `합들`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `합들`: 각 수의 모든 약수 합 (자기 자신 포함)

### 예시
`숫자들 = [6]` → `합들 = [12]` (1+2+3+6)
`숫자들 = [1, 6, 10]` → `합들 = [1, 12, 18]`
`숫자들 = [12]` → `합들 = [28]`

### 힌트
- 바깥 반복으로 리스트 순회, 안쪽 반복으로 약수의 합 계산
- 약수의 합 임시 변수는 직접 만들어 쓰세요
- `자료`의 `항목 추가하기` 블록으로 `합들`에 누적

<details>
<summary>자세한 접근법</summary>

- `i = 1..숫자들 항목 수` 반복:
  - `N = 숫자들의 i번째 항목`
  - 약수합 = `0`
  - `j = 1..N` 반복: `N mod j = 0` 이면 약수합에 `j` 더하기
  - `합들`에 약수합 추가

</details>

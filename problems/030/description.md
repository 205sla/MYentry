## 문제 설명

괄호 문자들로 이루어진 리스트 `괄호`를 받아 **짝이 올바르게 맞춰져 있는지** 판별해 말하기로 출력하세요. 괄호의 종류는 3가지: `(` `)`, `[` `]`, `{` `}`.

- 모든 괄호가 짝이 맞고 **중첩 순서까지** 올바르면 `올바름`
- 하나라도 어긋나면 `틀림`
- 빈 리스트는 올바른 것으로 간주 (`올바름`)

### 입력
- `괄호`: 괄호 문자 항목들의 리스트 (채점 시 자동 설정). 각 항목은 `"("`·`")"`·`"["`·`"]"`·`"{"`·`"}"` 중 하나

### 출력
- `올바름` 또는 `틀림`을 말하기

### 예시
입력: `["(", ")"]` → 출력: `올바름`
입력: `["(", "[", "]", ")"]` → 출력: `올바름`
입력: `[")", "("]` → 출력: `틀림`
입력: `["(", "]"]` → 출력: `틀림`

### 힌트
- 리스트 `괄호`는 **기본으로 제공**됩니다 (채점 시 값이 교체됨)
- 리스트 `스택`도 **기본으로 제공**됩니다 (처음엔 비어 있음)
- 변수 `현위치`, `현재`, `짝`, `틀림`은 **기본으로 제공**됩니다
- 핵심 아이디어: 리스트 `스택`을 **진짜 스택처럼** 쓰기
  - `현위치 = 1..괄호의 항목 수` 반복, `현재 = 괄호의 현위치번째 항목`
  - **여는 괄호** (`(`, `[`, `{`) 면 → `스택`에 `현재` 추가 (맨 뒤에 쌓기)
  - **닫는 괄호** (`)`, `]`, `}`) 면 → `스택`의 **맨 뒤 항목**과 짝이 맞는지 확인
    - `)`의 짝은 `(`, `]`의 짝은 `[`, `}`의 짝은 `{`
    - `스택`이 비어 있으면 `틀림 = 1`
    - 짝이 안 맞으면 `틀림 = 1`
    - 맞으면 `스택`의 맨 뒤 항목 삭제
- 반복이 끝난 뒤:
  - `틀림 = 1` 이거나 `스택의 항목 수 > 0` 이면 → `틀림` 말하기
  - 아니면 → `올바름` 말하기
- **조기 종료**: `틀림 = 1` 이 된 뒤에는 남은 반복을 무시해도 됩니다 (`틀림 = 0` 인 동안만 스택 작업)
- `자료` 카테고리의 `항목 추가하기`, `N번째 항목`, `N번째 항목 삭제하기`, `항목 수` 블록으로 스택 조작
- `판단` 카테고리의 `그리고` · `또는` · `=` 블록 활용

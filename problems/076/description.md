## 문제 설명

변수 `문장`을 **뒤집어서** 말하기로 출력하세요.

### 입력
- `문장`: 임의의 문자열 (채점 시 자동 설정)

### 출력
- `문장`을 역순으로 뒤집은 문자열 말하기

### 예시
`문장 = "hello"` → 출력: `olleh`
`문장 = "엔트리"` → 출력: `리트엔`
`문장 = "12345"` → 출력: `54321`

### 힌트
- `계산` 카테고리의 `X번째 글자` 블록으로 글자 하나씩 꺼내기
- `계산`의 `합치기` 블록으로 결과 문자열 누적
- `자료`의 `문자열 길이`(글자 수) 블록
- 결과 문자열과 위치 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 결과 변수를 `""`(빈 문자열)로 초기화
- 위치를 `문장의 길이`부터 `1`까지 줄이며 반복
  - 결과 = 결과 + `문장의 (위치)번째 글자`
- 반복 끝난 뒤 결과 말하기
- **주의**: 결과 변수는 반드시 빈 문자열로 초기화 (엔트리 기본값 `0`에 붙이면 앞에 `0`이 붙음)

</details>

## 문제 설명

리스트에서 사용자가 입력한 값이 몇 번째 위치에 있는지 찾아 말하는 프로그램을 만드세요.

### 입력
- `묻고 기다리기` 블록으로 찾으려는 값을 입력받습니다
- `리스트`: 탐색 대상 리스트 (채점 시 자동 설정)

### 출력
- 찾은 경우: `(값)은 리스트의 (위치)번 위치에 있습니다.` 형태로 말하기
- 못 찾은 경우: `입력하신 값이 리스트에 없습니다.` 로 말하기

### 예시
입력: `5`, 리스트: `[3, 1, 5, 7, 9]`
출력: `5은 리스트의 3번 위치에 있습니다.`

입력: `4`, 리스트: `[3, 1, 5, 7, 9]`
출력: `입력하신 값이 리스트에 없습니다.`

### 힌트
- `흐름` 카테고리의 반복문과 조건문을 사용하세요
- 리스트의 길이만큼 반복하며 각 위치의 값과 비교합니다
- 값을 찾았는지 여부를 기록하는 변수(`찾았다`)를 활용하세요

## 문제 설명

**파도반 수열** `P(N)`의 `N`번째 항을 구해 말하기로 출력하세요.

- `P(1) = P(2) = P(3) = 1`
- `N ≥ 4` 일 때 `P(N) = P(N-2) + P(N-3)`

수열: `1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, 16, 21, 28, 37, ...`

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- 파도반 수열의 `N`번째 항을 말하기

### 예시
`N = 1` → 출력: `1`
`N = 6` → 출력: `3`
`N = 12` → 출력: `16`

### 힌트
- 지금까지 계산한 항을 저장할 보조 리스트를 직접 만들어 쓰세요
- 점화식을 사용해 `i = 4..N` 까지 차례로 채워 나가기
- 피보나치와 달리 `N-2` 와 `N-3` 을 더함 (직전이 아닌 2칸·3칸 전)

<details>
<summary>자세한 접근법</summary>

- 보조 리스트에 `N+1`개 자리를 만든 후
  - 1번째·2번째·3번째 항목 모두 `1`로 세팅
- `i = 4..N` 반복:
  - `리스트의 i번째 항목 = (리스트의 (i-2)번째) + (리스트의 (i-3)번째)`
- 마지막에 `리스트의 N번째 항목` 말하기

</details>

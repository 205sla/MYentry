## 문제 설명

자연수 `N`의 **약수 개수**를 구해 말하기로 출력하세요. 약수는 `N`을 나머지 없이 나누는 자연수입니다.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `N`의 약수 개수를 말하기

### 예시
`N = 1` → 출력: `1` (약수: {1})
`N = 6` → 출력: `4` (약수: {1, 2, 3, 6})
`N = 12` → 출력: `6` (약수: {1, 2, 3, 4, 6, 12})
`N = 13` → 출력: `2` (소수: {1, 13})

### 힌트
- `흐름`의 `N번 반복` 블록으로 `1..N`을 순회
- `계산`의 `나누기의 나머지` 블록으로 나누어떨어지는지 판정
- 약수 개수를 셀 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 카운터 변수를 `0`으로 초기화
- `i = 1..N` 반복하며 `N mod i = 0` 이면 카운터 `+1`
- 반복 끝난 뒤 카운터 말하기

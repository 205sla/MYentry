## 문제 설명

두 변수 `단어1`과 `단어2`를 **이어 붙여** 한 문자열로 말하기로 출력하세요.

### 입력
- `단어1`: 첫 번째 문자열 (채점 시 자동 설정)
- `단어2`: 두 번째 문자열 (채점 시 자동 설정)

### 출력
- `단어1`과 `단어2`를 이어 붙인 문자열을 말하기

### 예시
`단어1 = 사과`, `단어2 = 바나나` → 출력: `사과바나나`
`단어1 = Hello`, `단어2 = World` → 출력: `HelloWorld`

### 힌트
- `계산` 카테고리의 `합치기` 블록
- `자료` 카테고리의 변수값 블록을 두 칸에 넣어 이어 붙이기

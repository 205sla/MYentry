## 문제 설명

`묻고 기다리기`로 입력받은 값 뒤에 `냥`을 붙여 말하세요.

### 입력
- `묻고 기다리기` 블록으로 임의의 문자열을 입력받습니다 (채점 시 자동 응답)

### 출력
- 입력받은 값에 `냥`을 이어 붙여 말하기

### 예시
입력: `반가워` → 출력: `반가워냥`
입력: `안녕` → 출력: `안녕냥`

### 힌트
- `자료` 카테고리의 `묻고 기다리기` 블록
- `계산` 카테고리의 `합치기` 블록
- `생김새` 카테고리의 `말하기` 블록

<details>
<summary>자세한 접근법</summary>

묻고 기다리기 블록으로 받은 값은 `대답`에 자동 저장됩니다.
`대답`과 `냥`을 `합치기` 블록으로 이어 붙여 말하기에 넣으면 끝.

</details>

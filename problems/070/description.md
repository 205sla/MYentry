## 문제 설명

변수 `문장`을 **1초간씩** `N`번 반복해 말하기로 출력하세요.

### 입력
- `문장`: 말할 문자열 (채점 시 자동 설정)
- `N`: 반복 횟수 (1 이상, 채점 시 자동 설정)

### 출력
- `문장`을 **1초간** 말하기 × `N`번

### 예시
`문장 = "안녕"`, `N = 3` → `안녕`, `안녕`, `안녕`
`문장 = "엔트리"`, `N = 1` → `엔트리`

### 힌트
- `흐름`의 `N번 반복` 블록
- `생김새`의 `X초 동안 말하기` 블록
- 반복 블록 안에 말하기 블록 하나만 넣으면 됩니다

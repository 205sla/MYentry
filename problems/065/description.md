## 문제 설명

비어 있는 리스트 `숫자들`에 `1`부터 `N`까지의 정수를 **순서대로** 추가하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)
- `숫자들`: 빈 리스트 (채점 시 자동 설정)

### 출력
- 리스트 `숫자들`: `[1, 2, ..., N]`

### 예시
`N = 3` → `숫자들 = [1, 2, 3]`
`N = 5` → `숫자들 = [1, 2, 3, 4, 5]`

### 힌트
- `흐름`의 `N번 반복` 블록
- `자료`의 `항목 추가하기` 블록
- 카운터 변수는 직접 만들어 쓰거나 반복 블록의 `i`를 활용

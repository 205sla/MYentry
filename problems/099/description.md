## 문제 설명

영어 **소문자**로만 이루어진 문자열 `문장`에서 각 알파벳(`a`~`z`)의 등장 횟수를 **크기 26의** 리스트 `빈도`에 차례로 담으세요.

- `빈도`의 1번째 항목 = `a`의 개수
- `빈도`의 2번째 항목 = `b`의 개수
- …
- `빈도`의 26번째 항목 = `z`의 개수

### 입력
- `문장`: 영어 소문자만 포함된 문자열 (채점 시 자동 설정)
- `빈도`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `빈도`: 길이 26의 정수 리스트. 각 알파벳의 빈도수

### 예시
`"abc"` → `빈도[0..25] = [1,1,1,0,0,...,0]`
`"aaab"` → `빈도[0..25] = [3,1,0,...,0]`

### 힌트
- 먼저 `빈도`를 `0`으로 26번 채우기
- `문장`의 각 글자에 대해 `a`로부터 몇 번째인지 계산해 해당 위치 +1
- 비교는 `a`~`z` 각각과 직접 비교하거나, 유니코드 차이 계산

<details>
<summary>자세한 접근법</summary>

- `빈도`에 `0`을 `26`번 추가해 초기화
- `문장`의 `i`번째 글자를 꺼내서 각 알파벳과 비교
  - `a`와 같으면 `빈도의 1번째 +1`
  - `b`와 같으면 `빈도의 2번째 +1`
  - …
- 또는 한 번의 글자 비교로 위치 인덱스를 구하는 방식도 가능

</details>

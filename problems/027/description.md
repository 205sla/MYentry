## 문제 설명

**오름차순으로 정렬된** 리스트 `숫자들`에서 **두 항목의 합이 `목표`와 같아지는 쌍**을 찾아 `작은값+큰값` 형식의 문자열로 말하기 출력하세요. 없으면 `없음`을 말하기.

정답 쌍이 둘 이상 존재할 수 있는 경우에는 **투 포인터 알고리즘**이 양 끝에서 안쪽으로 좁혀가며 먼저 만나는 쌍을 정답으로 합니다.

### 입력
- `숫자들`: **오름차순 정렬된** 정수 리스트 (채점 시 자동 설정)
- `목표`: 정수 (채점 시 자동 설정)

### 출력
- 찾은 쌍 `작은값+큰값` 또는 `없음`을 말하기

### 예시
입력: `숫자들 = [1,3,5,7,9]`, `목표 = 10` → 출력: `1+9`
입력: `숫자들 = [2,4,6,8]`, `목표 = 12` → 출력: `4+8`
입력: `숫자들 = [1,2,3,4,5]`, `목표 = 100` → 출력: `없음`

### 힌트
- 리스트 `숫자들`은 **기본으로 제공**됩니다 (채점 시 값이 교체됨)
- 변수 `목표`, `왼쪽`, `오른쪽`, `합`, `찾음`은 **기본으로 제공**됩니다
- 핵심 아이디어: 두 포인터가 **양 끝에서 안쪽으로** 좁혀간다
  - 초기값: `왼쪽 = 1`, `오른쪽 = 숫자들의 항목 수`, `찾음 = 0`
  - `왼쪽 < 오른쪽` **그리고** `찾음 = 0` 동안 반복:
    - `합 = 숫자들의 왼쪽번째 항목 + 숫자들의 오른쪽번째 항목`
    - 만약 `합 = 목표` → `찾음 = 1` (반복 종료 신호)
    - 그렇지 않고 `합 < 목표` → `왼쪽 = 왼쪽 + 1` (작은 쪽 포인터를 올려 합 키우기)
    - 그렇지 않으면 → `오른쪽 = 오른쪽 - 1` (큰 쪽 포인터를 내려 합 줄이기)
- 반복이 끝난 뒤:
  - `찾음 = 1` 이면 `(숫자들의 왼쪽번째 항목)과 "+"와 (숫자들의 오른쪽번째 항목) 합치기`를 말하기
  - 아니면 `없음`을 말하기
- 합치기 블록은 **두 번 중첩** 필요 (`A + "+" + B` 형태)
- `흐름` 카테고리의 `...인 동안 반복하기`, `판단` 카테고리의 `그리고`·`=`·`<`, `계산`의 `합치기` 블록을 활용하세요

## 문제 설명

자연수 `N`의 모든 **약수**를 오름차순으로 리스트 `약수들`에 담으세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)
- `약수들`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `약수들`: `N`의 약수들 (오름차순)

### 예시
`N = 1` → `약수들 = [1]`
`N = 6` → `약수들 = [1, 2, 3, 6]`
`N = 12` → `약수들 = [1, 2, 3, 4, 6, 12]`

### 힌트
- `흐름`의 `N번 반복` 블록으로 `1..N`을 차례로 살펴보기
- `계산`의 `나누기의 나머지` 블록으로 판정
- `자료`의 `항목 추가하기` 블록으로 약수를 `약수들`에 쌓기

<details>
<summary>자세한 접근법</summary>

- `i = 1..N` 반복
- `N mod i = 0` 이면 `약수들`에 `i`를 추가
- 오름차순으로 순회하므로 자연스럽게 오름차순 정렬됨

</details>

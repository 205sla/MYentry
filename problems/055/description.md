## 문제 설명

세 변수 `A`, `B`, `C` 중 **가장 큰 값**을 말하기로 출력하세요. 동률이어도 값 자체는 같으므로 그대로 출력합니다.

### 입력
- `A`, `B`, `C`: 정수 (채점 시 자동 설정)

### 출력
- 세 변수 중 최댓값을 말하기

### 예시
`A=3, B=7, C=5` → 출력: `7`
`A=9, B=9, C=1` → 출력: `9`

### 힌트
- `흐름`의 `만약 ... 이라면, 아니면` 블록과 `판단`의 `>` 블록
- 임시 최댓값 변수를 직접 만들어 쓰거나, 조건 중첩으로 해결

<details>
<summary>자세한 접근법</summary>

- `A > B` 이면 `A`와 `C` 중 큰 것, 아니면 `B`와 `C` 중 큰 것
- 또는 임시 변수에 `A`를 저장 후, `B`가 더 크면 교체, `C`가 더 크면 또 교체

</details>

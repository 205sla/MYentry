## 문제 설명

자연수 `N`을 `1`, `2`, `3`의 합으로 나타내는 방법의 **가짓수**를 구해 말하기로 출력하세요. 덧셈 순서가 다르면 서로 다른 방법으로 간주합니다.

예: `N = 4` → `4`가지
- `1+1+1+1`
- `1+1+2`, `1+2+1`, `2+1+1`
- `2+2`
- `1+3`, `3+1`
→ 총 `7가지`

(실제로 `N=4`는 `7가지`입니다. 아래 힌트의 점화식으로 확인할 수 있어요.)

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `N`을 1,2,3의 합으로 나타내는 가짓수를 말하기

### 예시
`N = 1` → 출력: `1`
`N = 2` → 출력: `2`
`N = 3` → 출력: `4`
`N = 4` → 출력: `7`
`N = 5` → 출력: `13`

### 힌트
- 점화식: `f(N) = f(N-1) + f(N-2) + f(N-3)` (N ≥ 4)
- 초깃값: `f(1) = 1`, `f(2) = 2`, `f(3) = 4`
- DP 리스트는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 보조 리스트에 `N+1`개 자리 준비
- 1·2·3번째 자리에 각각 `1`, `2`, `4` 저장
- `i = 4..N` 반복:
  - `리스트의 i번째 = (i-1번째) + (i-2번째) + (i-3번째)`
- 최종 `리스트의 N번째` 말하기

</details>

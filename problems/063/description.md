## 문제 설명

`N`부터 `1`까지 **역순**으로 **1초간씩** 차례로 말하기로 출력하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
1. `N`을 **1초간** 말하기
2. `N-1`을 **1초간** 말하기
3. …
4. `1`을 **1초간** 말하기

### 예시
`N = 3` → `3`, `2`, `1`을 차례로 말하기
`N = 5` → `5`, `4`, `3`, `2`, `1`

### 힌트
- `흐름`의 `N번 반복` 블록
- 현재 말할 값을 담을 카운터 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 카운터를 `N`으로 초기화
- `N`번 반복하며 카운터를 1초간 말하기 한 뒤 카운터 `-1`
- 또는 `i = 1..N`으로 반복하며 `(N - i + 1)`을 계산해 말하기

</details>

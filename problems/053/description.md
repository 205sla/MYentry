## 문제 설명

변수 `N`이 **양수**·**음수**·**0** 중 어느 것인지 판별해 말하기로 출력하세요.

### 입력
- `N`: 정수 (채점 시 자동 설정)

### 출력
- `N > 0` 이면 `양수`
- `N < 0` 이면 `음수`
- `N = 0` 이면 `0`

### 예시
`N = 5` → 출력: `양수`
`N = -3` → 출력: `음수`
`N = 0` → 출력: `0`

### 힌트
- `흐름` 카테고리의 `만약 ... 이라면, 아니면` 블록을 중첩하거나 이어서 쓰기
- `판단` 카테고리의 `>`, `<`, `=` 블록 활용

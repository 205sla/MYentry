## 문제 설명

변수 `점수`에 따라 학점을 판정해 말하기로 출력하세요.

- `90 ~ 100` → `A`
- `80 ~ 89` → `B`
- `70 ~ 79` → `C`
- `60 ~ 69` → `D`
- `0 ~ 59` → `F`

### 입력
- `점수`: 0 이상 100 이하의 정수 (채점 시 자동 설정)

### 출력
- 학점 (`A`·`B`·`C`·`D`·`F` 중 하나) 말하기

### 예시
`점수 = 95` → 출력: `A`
`점수 = 72` → 출력: `C`
`점수 = 59` → 출력: `F`

### 힌트
- `흐름`의 `만약 ... 이라면, 아니면` 블록을 중첩하거나 이어서 쓰기
- `판단`의 `≥` 블록(또는 `>`·`=` 조합)

<details>
<summary>자세한 접근법</summary>

- 점수가 90 이상이면 `A`
- 아니고 80 이상이면 `B`
- 아니고 70 이상이면 `C`
- 아니고 60 이상이면 `D`
- 아니면 `F`

</details>

## 문제 설명

두 변수 `변수1`과 `변수2`의 **평균**을 말하기로 출력하세요.

(모든 테스트 케이스는 두 수의 합이 짝수라서 평균이 정수가 됩니다.)

### 입력
- `변수1`: 첫 번째 정수 (채점 시 자동 설정)
- `변수2`: 두 번째 정수 (채점 시 자동 설정)

### 출력
- `(변수1 + 변수2) ÷ 2`를 말하기

### 예시
변수1: `4`, 변수2: `6` → 출력: `5`
변수1: `10`, 변수2: `20` → 출력: `15`

### 힌트
- `계산` 카테고리의 덧셈 블록으로 두 수를 합한 뒤
- 나누기 블록으로 `2`로 나누기
- `생김새`의 `말하기` 블록으로 결과 출력

## 문제 설명

피보나치 수열 `1, 1, 2, 3, 5, 8, 13, 21, ...` 중 **`M` 이하**인 항이 몇 개인지 세어 말하기로 출력하세요.

- `F(1) = 1`, `F(2) = 1`, `F(N) = F(N-1) + F(N-2)` (N ≥ 3)
- 같은 값이 두 번 나와도 각각 세어집니다 (예: `M ≥ 1`이면 `1`이 두 번 포함)

### 입력
- `M`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `M` 이하 피보나치 항의 개수를 말하기

### 예시
`M = 10` → 출력: `6` (1, 1, 2, 3, 5, 8)
`M = 20` → 출력: `7` (1,1,2,3,5,8,13)
`M = 1` → 출력: `2` (1, 1)

### 힌트
- 직전 두 항을 담을 보조 변수와 카운터 변수는 직접 만들어 쓰세요
- `...인 동안 반복`으로 새 항이 `M` 이하인 동안 계속 생성

<details>
<summary>자세한 접근법</summary>

- 이전·현재를 `1`, `1`로 시작. 카운터를 `0`.
- 현재가 `M` 이하인 동안:
  - 카운터 +1
  - 다음 항 = 이전 + 현재
  - 이전 = 현재, 현재 = 다음 항
- 반복 종료 후 카운터 말하기

</details>

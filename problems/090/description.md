## 문제 설명

`1` 이상 `N` 이하의 모든 **완전수**를 리스트 `완전수들`에 오름차순으로 담으세요.

**완전수**: 자기 자신을 제외한 모든 약수의 합이 자기 자신과 같은 수.
- `6 = 1 + 2 + 3` ✓
- `28 = 1 + 2 + 4 + 7 + 14` ✓
- `496 = 1 + 2 + 4 + 8 + 16 + 31 + 62 + 124 + 248`

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)
- `완전수들`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `완전수들`: `N` 이하의 완전수 목록 (오름차순)

### 예시
`N = 10` → `완전수들 = [6]`
`N = 100` → `완전수들 = [6, 28]`
`N = 5` → `완전수들 = []`

### 힌트
- 바깥 반복 `i = 1..N`에서 각 `i`가 완전수인지 판정
- 안쪽 반복으로 `i`의 진약수(자기 자신 제외)의 합을 구해 `i`와 비교
- 진약수 합 변수·약수 후보 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- `i = 1..N` 반복
  - 진약수합을 `0`으로 초기화
  - `j = 1..(i-1)` 반복: `i mod j = 0` 이면 진약수합에 `j` 더하기
  - 진약수합 = `i` 이면 `완전수들`에 `i` 추가

</details>

## 문제 설명

`리스트`에 들어 있는 모든 숫자의 **합**을 구해 말하기로 출력하세요.

### 입력
- `리스트`: 합을 구할 숫자들 (채점 시 자동 설정)

### 출력
- 모든 항목의 합을 말하기

### 예시
입력: `[1, 2, 3]` → 출력: `6`
입력: `[5, 10, 15, 20]` → 출력: `50`
입력: `[]` → 출력: `0`

### 힌트
- `흐름` 카테고리의 반복 블록으로 리스트 길이만큼 돌기
- `자료` 카테고리의 `N번째 항목` 블록으로 값 꺼내기
- 누적할 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 합을 담을 변수를 `0`으로 초기화
- 리스트 길이만큼 반복하며 각 항목을 더하기
- 반복이 끝나면 최종 값을 말하기
- 빈 리스트라면 더할 항목이 없으니 `0`이 그대로 출력됨

</details>

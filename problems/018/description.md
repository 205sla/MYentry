## 문제 설명

`리스트`에 들어 있는 모든 숫자들의 **합**을 구해서 말하기로 출력하세요.

### 입력
- `리스트`: 합을 구할 숫자들 (채점 시 자동 설정)

### 출력
- 모든 항목의 합을 말하기

### 예시
입력: `[1, 2, 3]` → 출력: `6`
입력: `[5, 10, 15, 20]` → 출력: `50`

### 힌트
- 풀이를 시작하기 전에 `자료` 카테고리에서 **리스트 `리스트`** 와 **변수 `합계`** 를 만들어 주세요
- `흐름` 카테고리의 `N번 반복하기`와 `자료` 카테고리의 `리스트 항목 수` 블록을 함께 쓰면 리스트 길이만큼 반복할 수 있어요
- `리스트의 N번째 항목` 블록으로 값을 하나씩 꺼내 `합계`에 더해가세요
- 반복이 끝나면 `합계` 변수를 말하기로 출력하세요

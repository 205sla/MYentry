## 문제 설명

자연수 `N`의 **자릿수를 뒤집은** 값을 **숫자**로 말하기로 출력하세요. 앞쪽에 오던 `0`은 자연스럽게 사라집니다.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `N`의 자릿수를 역순으로 배치한 숫자를 말하기

### 예시
`N = 12345` → 출력: `54321`
`N = 100` → 출력: `1` (`001`의 앞쪽 `0` 소거)
`N = 7` → 출력: `7`
`N = 1200` → 출력: `21`

### 힌트
- 한 자릿수씩 `10으로 나눈 나머지`로 떼어내고, `결과 = 결과 × 10 + 자리` 로 누적
- `10으로 나눈 몫`으로 남은 수에서 마지막 자리를 떼어냄
- 결과·남은 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 결과 = `0`, 남은 = `N`
- 남은 > 0 인 동안:
  - 자리 = 남은 mod 10
  - 결과 = 결과 × 10 + 자리
  - 남은 = 남은 ÷ 10
- 반복 종료 후 결과 말하기

</details>

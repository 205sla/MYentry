## 문제 설명

문자열 `문장`을 공백(`" "`) 기준으로 나눴을 때 **단어의 개수**를 말하기로 출력하세요.

- 앞·뒤·중간의 연속 공백도 하나의 구분으로 간주 (빈 단어는 세지 않음)
- 빈 문자열 또는 공백만 있는 문자열 → `0`

### 입력
- `문장`: 임의의 문자열 (채점 시 자동 설정)

### 출력
- 단어 개수를 말하기

### 예시
`"hello world"` → `2`
`"  a  b  "` → `2`
`"hello"` → `1`
`""` → `0`

### 힌트
- 현재 글자와 직전 글자를 비교해 "공백→비공백" 전환에서만 +1
- 이전 글자가 공백이었는지 기록하는 플래그 변수가 필요
- 카운터·플래그 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 카운터를 `0`, 이전공백 플래그를 `1` (처음은 공백으로 간주)
- 문장을 한 글자씩 순회:
  - 현재 글자가 공백이면 이전공백 = `1`
  - 아니면 이전공백 = `1` 일 때만 카운터 +1, 그 후 이전공백 = `0`
- 반복 종료 후 카운터 말하기

</details>

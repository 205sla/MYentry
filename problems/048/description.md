## 문제 설명

두 변수 `변수1`을 `변수2`로 나눈 **나머지**를 말하기로 출력하세요.

### 입력
- `변수1`: 나뉨수 정수 (채점 시 자동 설정)
- `변수2`: 나눔수 정수, `0`이 아님 (채점 시 자동 설정)

### 출력
- `변수1`을 `변수2`로 나눈 나머지를 말하기

### 예시
변수1: `10`, 변수2: `3` → 출력: `1`
변수1: `20`, 변수2: `4` → 출력: `0`

### 힌트
- `계산` 카테고리의 `나누기의 나머지` 블록 사용
- `자료`의 변수값 블록, `생김새`의 `말하기` 블록과 조합

## 문제 설명

정수 `N`에 다음 세 연산을 적용해 **`1`로 만드는 최소 횟수**를 구해 말하기로 출력하세요.

- `3`으로 나누어떨어지면 `3`으로 나눈다
- `2`로 나누어떨어지면 `2`로 나눈다
- `1`을 뺀다

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `N`을 `1`로 만드는 최소 연산 횟수를 말하기

### 예시
`N = 1` → 출력: `0`
`N = 2` → 출력: `1`
`N = 10` → 출력: `3` (10→9→3→1)
`N = 6` → 출력: `2` (6→3→1 또는 6→2→1)

### 힌트
- DP: `dp[i]`는 `i`를 `1`로 만드는 최소 횟수
- `dp[1] = 0`
- `dp[i] = dp[i-1] + 1` 을 기본으로, `2`·`3` 배수면 `dp[i/2]`·`dp[i/3]`도 후보
- DP 리스트는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- `dp` 리스트에 `N + 1`개 항목을 `0` 으로 채움
- `i = 2..N` 반복:
  - `dp[i] = dp[i-1] + 1`  (기본: -1 연산)
  - `i mod 2 = 0` 이고 `dp[i/2] + 1 < dp[i]` → `dp[i] = dp[i/2] + 1`
  - `i mod 3 = 0` 이고 `dp[i/3] + 1 < dp[i]` → `dp[i] = dp[i/3] + 1`
- `dp[N]` 말하기

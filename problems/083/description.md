## 문제 설명

두 리스트 `A`와 `B`를 이어 붙인 결과를 리스트 `결과`에 담으세요. (`A` 먼저, `B` 다음)

### 입력
- `A`: 첫 번째 리스트 (채점 시 자동 설정)
- `B`: 두 번째 리스트 (채점 시 자동 설정)
- `결과`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `결과`: `A` 항목들 + `B` 항목들 (순서 유지)

### 예시
`A = [1, 2]`, `B = [3, 4, 5]` → `결과 = [1, 2, 3, 4, 5]`
`A = []`, `B = [10]` → `결과 = [10]`
`A = [7]`, `B = []` → `결과 = [7]`

### 힌트
- `A`를 순서대로 순회하며 `결과`에 추가
- 그 다음 `B`도 같은 방식
- `자료`의 `항목 추가하기`·`항목 수`·`N번째 항목` 블록

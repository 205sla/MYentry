## 문제 설명

`1`부터 `N`까지의 **짝수만** 골라 합을 구해 말하기로 출력하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- 1~N 범위의 짝수들의 합

### 예시
`N = 5` → 출력: `6` (2+4)
`N = 10` → 출력: `30` (2+4+6+8+10)
`N = 1` → 출력: `0` (짝수 없음)

### 힌트
- `흐름`의 `N번 반복` 블록
- `계산`의 `나누기의 나머지` + `판단`의 `=` 블록으로 짝수 판정
- 합 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 합 변수를 `0`으로 초기화
- `i = 1..N` 반복하며 `i mod 2 = 0` 이면 합에 `i`를 더하기
- 반복 끝난 뒤 합 말하기

</details>

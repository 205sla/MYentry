## 문제 설명

리스트 `숫자들`의 **평균보다 큰** 항목이 몇 개인지 세어 말하기로 출력하세요.

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 평균 초과 항목의 개수를 말하기

### 예시
`[1, 2, 3, 4, 5]` (평균 3) → 출력: `2` (4, 5)
`[7, 7, 7]` → 출력: `0` (모두 평균과 같음)
`[1, 10]` (평균 5.5) → 출력: `1` (10)

### 힌트
- 두 번 순회: 평균 계산 → 평균과 각 항목 비교
- 합·평균·카운터 변수는 직접 만들어 쓰세요
- `판단`의 `>` 블록

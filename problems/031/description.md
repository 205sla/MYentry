## 문제 설명

**행복수(Happy Number)**: 자연수 `N`의 각 자릿수를 제곱해 더하는 과정을 반복하면 결국 `1`에 도달하는 수를 **행복수**라고 부릅니다. 반대로 과정이 이미 지나온 값으로 돌아와 무한 순환에 빠지면 **불행수**입니다.

예시 (19는 행복수):
- 19 → 1² + 9² = 82
- 82 → 8² + 2² = 68
- 68 → 6² + 8² = 100
- 100 → 1² + 0² + 0² = **1** ✓

예시 (4는 불행수 — 자기 자신으로 돌아옴):
- 4 → 16 → 37 → 58 → 89 → 145 → 42 → 20 → 4 → …

`N`이 행복수면 `행복수`, 아니면 `불행수`를 말하기로 출력하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `행복수` 또는 `불행수`를 말하기

### 예시
입력: `N = 1` → 출력: `행복수`
입력: `N = 19` → 출력: `행복수`
입력: `N = 4` → 출력: `불행수`

### 힌트
- 변수 `N`, `합`, `남은`, `자리`, `보임`, `결과`는 **기본으로 제공**됩니다
- 리스트 `이력`도 **기본으로 제공**됩니다 (지금까지 거쳐온 값 저장용)
- ⚠️ 변수 `결과`는 엔트리 기본값 `0`으로 시작합니다 — 반복 시작 전에 반드시 **빈 문자열 `""`로 초기화**하세요
- 바깥 반복: `결과 = ""` 인 동안 반복
  - `N = 1` 이면 → `결과 = "행복수"`로 설정 (반복 탈출)
  - 아니면, `N`이 `이력`에 이미 있는지 검사 (`보임` 변수 사용):
    - `보임 = 0`으로 시작, `이력`을 순회하면서 각 항목과 `N`을 비교해 같으면 `보임 = 1`
    - `보임 = 1` 이면 → `결과 = "불행수"`로 설정 (순환 감지 → 탈출)
    - 아니면 → `N`을 `이력`에 추가하고, **자릿수 제곱합**을 계산해서 `N`을 갱신
- 안쪽 반복 (자릿수 제곱합):
  - `합 = 0`, `남은 = N`
  - `남은 > 0` 동안 반복:
    - `자리 = 남은을 10으로 나눈 나머지`
    - `합 = 합 + 자리 × 자리`
    - `남은 = 남은을 10으로 나눈 몫`
  - 반복 후 `N = 합`
- 바깥 반복이 끝나면 `결과`를 말하기
- `흐름` 카테고리의 `...인 동안 반복하기`·`N번 반복하기`, `계산`의 `나누기의 몫`·`나머지`·`곱하기`, `자료`의 `항목 추가하기` 블록 활용

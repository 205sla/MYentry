## 문제 설명

변수 `N`이 **홀수**인지 **짝수**인지 판별해 말하기로 출력하세요.

### 입력
- `N`: 정수 (채점 시 자동 설정)

### 출력
- 홀수이면 `홀수`를 말하기
- 짝수이면 `짝수`를 말하기

### 예시
`N = 3` → 출력: `홀수`
`N = 8` → 출력: `짝수`
`N = 0` → 출력: `짝수`
`N = -5` → 출력: `홀수`

### 힌트
- `계산` 카테고리의 `나누기의 나머지` 블록으로 `N`을 `2`로 나눈 나머지 구하기
- `흐름` 카테고리의 `만약 ... 이라면, 아니면` 블록으로 `0`과 비교
- `판단` 카테고리의 `=` 블록 활용

<details>
<summary>자세한 접근법</summary>

- `N mod 2 = 0` 이면 `짝수` 말하기
- 아니면 `홀수` 말하기
- 음수도 나머지 규칙으로 판정됨 (-4는 짝수, -3은 홀수)

</details>

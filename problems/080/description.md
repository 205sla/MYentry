## 문제 설명

별(`*`)을 차곡차곡 쌓아 직각삼각형 모양을 `N`줄 **1초간씩** 차례로 말하기로 출력하세요. `i`번째 줄에는 `i`개의 별이 있어야 합니다.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
1. `*` (1개)
2. `**` (2개)
3. `***` (3개)
4. …
N. `*` × N개

모든 말하기는 **1초간**.

### 예시
`N = 3` →
- 1초간 `*`
- 1초간 `**`
- 1초간 `***`

### 힌트
- 바깥 반복 `i = 1..N`
- 안쪽 반복으로 `i`개의 `*`를 `합치기` 블록으로 붙여 한 줄 만들기
- 현재 줄 문자열 변수와 카운터 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 바깥: `i = 1..N` 반복
  - 줄 변수를 `""`(빈 문자열)로 초기화
  - 안쪽: `i`번 반복하며 `줄 = 줄 + "*"`
  - 완성된 줄을 1초간 말하기

</details>

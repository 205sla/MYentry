## 문제 설명

리스트 `숫자들`의 **평균**(합 ÷ 항목 수)을 말하기로 출력하세요.

모든 테스트 케이스는 평균이 정수가 나오도록 구성되어 있습니다.

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 평균을 말하기 (정수)

### 예시
`[1, 2, 3]` → 출력: `2`
`[10, 20, 30, 40, 50]` → 출력: `30`
`[5]` → 출력: `5`

### 힌트
- 리스트 합을 누적한 뒤 `자료`의 `항목 수`로 나누기
- 합 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 합 변수를 `0`으로 초기화
- 리스트 순회하며 각 항목을 합에 더하기
- 평균 = 합 ÷ 리스트 항목 수

</details>

## 문제 설명

`리스트`에 들어 있는 숫자들 중 **홀수의 개수**와 **짝수의 개수**를 각각 세어 말하기로 출력하세요.

### 입력
- `리스트`: 개수를 셀 숫자들 (채점 시 자동 설정)

### 출력
- 홀수의 개수를 말하기
- 짝수의 개수를 말하기

### 예시
입력: `[1, 2, 3, 4, 5]` → 홀수 `3`, 짝수 `2`
입력: `[2, 4, 6, 8]` → 홀수 `0`, 짝수 `4`

### 힌트
- `흐름` 카테고리의 반복 블록으로 리스트 순회
- `계산` 카테고리의 `나누기의 나머지` 블록 + `판단`의 `=` 블록으로 홀짝 판정
- 홀수와 짝수 개수를 셀 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 홀수·짝수 카운터 변수를 각각 `0`으로 초기화
- 리스트 길이만큼 반복하며 각 항목을 `2`로 나눈 나머지 확인
- 나머지 `0` → 짝수 카운터 +1, 아니면 홀수 카운터 +1
- 음수도 나머지 규칙으로 판정 (-4는 짝수, -3은 홀수)

</details>

## 문제 설명

`리스트`에 들어 있는 숫자들 중 **홀수의 개수**와 **짝수의 개수**를 각각 세어 말하기로 출력하세요.

### 입력
- `리스트`: 개수를 셀 숫자들 (채점 시 자동 설정)

### 출력
- 홀수의 개수를 말하기
- 짝수의 개수를 말하기

### 예시
입력: `[1, 2, 3, 4, 5]` → 홀수 `3`, 짝수 `2`
입력: `[2, 4, 6, 8]` → 홀수 `0`, 짝수 `4`

### 힌트
- 리스트 `리스트`와 변수 `홀수`, `짝수`는 **기본으로 제공**됩니다
- `흐름` 카테고리의 `N번 반복하기`와 `자료` 카테고리의 `리스트 항목 수`로 순회하세요
- `계산` 카테고리의 `나머지` 블록으로 `2`로 나눈 나머지를 확인하고, `판단` 카테고리의 `=` 블록으로 0과 비교하세요
- 나머지가 0이면 짝수, 아니면 홀수입니다
- 0은 짝수이고, 음수도 홀수/짝수가 있어요 (-4는 짝수, -3은 홀수)

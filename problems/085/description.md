## 문제 설명

두 사람이 낸 가위·바위·보를 비교해 승자를 판정하세요.

- 두 사람이 같은 걸 내면: `무승부`
- A가 이기면: `A 승`
- B가 이기면: `B 승`

가위·바위·보 규칙:
- 가위는 보를 이김
- 바위는 가위를 이김
- 보는 바위를 이김

### 입력
- `A`, `B`: 문자열 `"가위"` / `"바위"` / `"보"` 중 하나 (채점 시 자동 설정)

### 출력
- `무승부` / `A 승` / `B 승` 중 하나를 말하기

### 예시
`A = "가위"`, `B = "보"` → 출력: `A 승`
`A = "바위"`, `B = "가위"` → 출력: `A 승`
`A = "보"`, `B = "보"` → 출력: `무승부`

### 힌트
- `판단`의 `=` 블록으로 두 값 비교해 무승부 먼저 처리
- 그 다음 A가 이기는 3가지 조합을 `또는` 으로 묶어 판정
- 나머지는 B 승

<details>
<summary>자세한 접근법</summary>

- `A = B` 이면 `무승부` 출력 후 종료
- `A 이기는 세 조합` 중 하나라도 맞으면 `A 승`:
  - `A = "가위"` 그리고 `B = "보"`
  - `A = "바위"` 그리고 `B = "가위"`
  - `A = "보"` 그리고 `B = "바위"`
- 아니면 `B 승`

</details>

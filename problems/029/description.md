## 문제 설명

2 이상의 자연수 `N`이 주어질 때, `N` 이하의 **모든 소수**를 **에라토스테네스의 체** 방법으로 구해 리스트 `소수들`에 오름차순으로 담으세요.

에라토스테네스의 체란? 2부터 차례로 보면서 "소수라면 그 배수를 전부 지워나가는" 방식입니다. 마지막까지 지워지지 않은 수가 소수입니다.

### 입력
- `N`: 2 이상의 자연수 (채점 시 자동 설정)

### 출력
- 리스트 `소수들`: `N` 이하의 모든 소수 (오름차순)

### 예시
입력: `N = 10` → `소수들 = [2, 3, 5, 7]`
입력: `N = 20` → `소수들 = [2, 3, 5, 7, 11, 13, 17, 19]`
입력: `N = 2` → `소수들 = [2]`

### 힌트
- 변수 `N`, `i`, `j`는 **기본으로 제공**됩니다
- 리스트 `체`, `소수들`도 **기본으로 제공**됩니다 (둘 다 처음엔 비어 있음)
- 핵심 아이디어: 크기 `N`짜리 깃발 리스트 `체`를 만들어 **배수를 지워나가기**
  - **1단계 (초기화)**: `체`를 비우고, `i = 1..N`으로 반복하며 `1`을 `N`번 추가 (전체를 소수 후보로 표시)
  - **2단계 (지우기)**: `i = 2..N` 반복
    - 만약 `체의 i번째 항목 = 1` 이면 (아직 소수 후보라면):
      - `j`를 `i + i`부터 시작해 `j <= N` 인 동안 `i`씩 늘려가면서
      - `체의 j번째 항목 = 0`으로 설정 (i의 배수 지우기)
  - **3단계 (수집)**: `i = 2..N` 반복
    - 만약 `체의 i번째 항목 = 1` 이면 `소수들`에 `i` 추가
- 엔트리의 `N번 반복하기` / `...인 동안 반복하기`, 리스트의 `항목 추가하기` · `N번째 항목을 X로 바꾸기`, 계산의 `더하기` 블록을 활용하세요

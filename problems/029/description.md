## 문제 설명

2 이상의 자연수 `N`이 주어질 때, `N` 이하의 **모든 소수**를 **에라토스테네스의 체** 방법으로 구해 리스트 `소수들`에 오름차순으로 담으세요.

에라토스테네스의 체란? 2부터 차례로 보면서 "소수라면 그 배수를 전부 지워나가는" 방식입니다. 마지막까지 지워지지 않은 수가 소수입니다.

### 입력
- `N`: 2 이상의 자연수 (채점 시 자동 설정)

### 출력
- 리스트 `소수들`: `N` 이하의 모든 소수 (오름차순)

### 예시
입력: `N = 10` → `소수들 = [2, 3, 5, 7]`
입력: `N = 20` → `소수들 = [2, 3, 5, 7, 11, 13, 17, 19]`
입력: `N = 2` → `소수들 = [2]`

### 힌트
- `흐름`의 `N번 반복하기`·`...인 동안 반복하기` 중첩
- `자료`의 `항목 추가하기`·`N번째 항목을 X로 바꾸기`
- 깃발 역할 리스트와 순회용 보조 변수는 직접 만들어 쓰세요 (2차원 없이 1D 리스트 1개로 해결)

<details>
<summary>자세한 접근법</summary>

- **1단계 (초기화)**: 크기 `N`짜리 깃발 리스트를 만들고 `1`을 `N`번 채움 (모두 소수 후보로 시작)
- **2단계 (지우기)**: `i = 2..N`으로 반복
  - 깃발의 `i`번째 항목이 `1`이면 (아직 후보라면):
    - `j`를 `i + i`부터 `j ≤ N`인 동안 `i`씩 증가시키며
    - 깃발의 `j`번째 항목을 `0`으로 바꿈 (`i`의 배수 지우기)
- **3단계 (수집)**: `i = 2..N`으로 반복하며 깃발이 `1`인 `i`를 `소수들`에 추가

</details>

## 문제 설명

가운데 정렬된 **피라미드** 모양으로 별(`*`)을 `N`줄 **1초간씩** 차례로 말하기로 출력하세요.

- `i`번째 줄: 앞에 공백 `(N - i)`개 + 별 `(2i - 1)`개

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
`N = 3` 인 경우:
1. `  *` (공백 2개 + 별 1개)
2. ` ***` (공백 1개 + 별 3개)
3. `*****` (공백 0개 + 별 5개)

모든 말하기는 **1초간**.

### 예시
`N = 3` → `  *`, ` ***`, `*****`
`N = 1` → `*`

### 힌트
- 바깥 반복 `i = 1..N`
- 안쪽에서 공백 `(N-i)`개와 별 `(2i-1)`개를 붙여 한 줄 생성
- 줄 변수·카운터 변수는 직접 만들어 쓰세요

<details>

- 바깥: `i = 1..N`

  - `(2i - 1)`번 반복하며 줄에 `*` 하나씩 추가
  - 완성된 줄을 1초간 말하기

## 문제 설명

변수 `단`의 구구단을 9줄 **1초간씩** 차례로 말하기로 출력하세요. 형식은 `(단) x (i) = (곱)`.

### 입력
- `단`: 2 이상 9 이하의 자연수 (채점 시 자동 설정)

### 출력
1. `단 x 1 = ...`을 **1초간** 말하기
2. `단 x 2 = ...`을 **1초간** 말하기
3. …
9. `단 x 9 = ...`을 **1초간** 말하기

### 예시
`단 = 3` → `3 x 1 = 3`, `3 x 2 = 6`, …, `3 x 9 = 27` 차례로

### 힌트
- `흐름`의 `N번 반복` 블록
- `계산`의 `합치기` 블록을 여러 번 중첩해 `단 + " x " + i + " = " + (단 × i)` 형태 생성
- 카운터 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- `i = 1..9` 반복
- 각 줄마다 `(단) x (i) = (단 × i)` 문자열을 만들고 1초간 말하기
- 곱은 `단 × i`로 계산

</details>

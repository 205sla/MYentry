## 문제 설명

`묻고 기다리기` 블록으로 찾으려는 값을 입력받고, `리스트`에서 해당 값의 위치(1부터 시작)를 찾아 `결과` 변수에 저장하세요. 값을 찾지 못하면 `결과`에 `0`을 저장하세요.

### 입력
- `묻고 기다리기` 블록으로 찾으려는 값을 입력받습니다
- `리스트`: 탐색 대상 리스트 (채점 시 자동 설정)

### 출력
- `결과`: 찾은 위치 (1부터 시작). 못 찾으면 `0`

### 예시
입력: `5`, 리스트: `[3, 1, 5, 7, 9]`
결과: `3`

입력: `4`, 리스트: `[3, 1, 5, 7, 9]`
결과: `0`

### 힌트
- 리스트의 길이만큼 반복하며 각 위치의 값과 `대답`을 비교합니다
- 일치하면 해당 위치를 `결과`에 저장하고, `찾았다`를 1로 설정합니다
- 반복이 끝난 후 `찾았다`가 0이면 `결과`에 0을 저장합니다

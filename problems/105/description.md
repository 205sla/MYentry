## 문제 설명

달팽이가 높이 `H`미터 우물의 바닥에서 올라갑니다.
- 낮에 `A`미터 올라감
- 밤에 `B`미터 미끄러짐 (단, 이미 우물 밖으로 나왔으면 미끄러지지 않음)

며칠 만에 우물 밖으로 완전히 나오는지 말하기로 출력하세요.

보장: `A > B`, `A < H` 가정할 필요 없이 모든 입력에 대해 계산할 수 있어야 합니다.

### 입력
- `H`: 우물 높이 (1 이상)
- `A`: 낮에 올라가는 거리
- `B`: 밤에 미끄러지는 거리 (0 이상, `A`보다 작음)
(채점 시 자동 설정)

### 출력
- 며칠 만에 우물 밖으로 나오는지 말하기

### 예시
`H=10, A=2, B=1` → 출력: `9`
`H=10, A=3, B=1` → 출력: `5`
`H=5, A=10, B=1` → 출력: `1`

### 힌트
- 시뮬레이션: 매일 아침 `A` 올라간 뒤 우물 밖(높이 이상)인지 판정
- 나오지 못했다면 `B` 미끄러지고 다음 날
- 현재 위치·일수 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법 (시뮬레이션)</summary>

- 현재 = `0`, 일 = `0`
- `현재 + A < H` 인 동안 반복:
  - 일 += 1
  - 현재 += `A - B` (낮에 올라가고 밤에 미끄러진 총합)
- 반복 종료 시점에는 다음 날 `A`만큼 올라가면 우물 밖으로 나옴
- 총 일수 = 반복 횟수 + `1`

</details>

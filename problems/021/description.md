## 문제 설명

두 자연수 `a`와 `b`의 **최대공약수**를 **유클리드 호제법**으로 구해 말하기로 출력하세요.

유클리드 호제법은 두 수의 최대공약수를 빠르게 구하는 방법입니다.
- `a`를 `b`로 나눈 나머지를 `r`이라 하면, `a`와 `b`의 최대공약수는 `b`와 `r`의 최대공약수와 같습니다.
- 이를 반복하다가 나머지가 `0`이 되면 그때의 `b`가 최대공약수입니다.

### 입력
- `a`, `b`: 두 자연수 (채점 시 자동 설정, `1` 이상)

### 출력
- `a`와 `b`의 최대공약수를 말하기

### 예시
입력: `a = 12`, `b = 18` → 최대공약수 `6`
입력: `a = 48`, `b = 36` → 최대공약수 `12`
입력: `a = 17`, `b = 5` → 최대공약수 `1`

### 힌트
- 변수 `a`, `b`는 **기본으로 제공**됩니다 — 채점 시 자동으로 값이 설정됩니다
- 풀이에 필요한 변수 `나머지`도 미리 만들어 제공됩니다
- `흐름` 카테고리의 `...인 동안 반복하기` 블록으로 `b`가 `0`이 아닌 동안 반복하세요
- `계산` 카테고리의 `나누기의 나머지` 블록으로 `a`를 `b`로 나눈 나머지를 구합니다
- 반복 안에서 `나머지`에 `a를 b로 나눈 나머지`를 저장하고, `a`에 `b`를, `b`에 `나머지`를 차례로 대입하세요
- 반복이 끝나면 `a`가 최대공약수입니다 — `생김새` 카테고리의 `말하기`로 출력하세요

## 문제 설명

두 자연수 `a`와 `b`의 **최대공약수**를 **유클리드 호제법**으로 구해 말하기로 출력하세요.

유클리드 호제법은 두 수의 최대공약수를 빠르게 구하는 방법입니다.
- `a`를 `b`로 나눈 나머지를 `r`이라 하면, `a`와 `b`의 최대공약수는 `b`와 `r`의 최대공약수와 같습니다.
- 이를 반복하다가 나머지가 `0`이 되면 그때의 `b`가 최대공약수입니다.

### 입력
- `a`, `b`: 두 자연수 (채점 시 자동 설정, `1` 이상)

### 출력
- `a`와 `b`의 최대공약수를 말하기

### 예시
입력: `a = 12`, `b = 18` → 최대공약수 `6`
입력: `a = 48`, `b = 36` → 최대공약수 `12`
입력: `a = 17`, `b = 5` → 최대공약수 `1`

### 힌트
- `흐름` 카테고리의 `...인 동안 반복` 블록으로 `b`가 `0`이 아닌 동안 반복
- `계산` 카테고리의 `나누기의 나머지` 블록
- 나머지를 잠시 담아둘 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- `b`가 `0`이 아닌 동안 반복:
  - 보조 변수에 `a`를 `b`로 나눈 나머지를 저장
  - `a`에 `b`를 대입
  - `b`에 보조 변수(나머지) 값을 대입
- 반복이 끝났을 때 `a`가 최대공약수

</details>

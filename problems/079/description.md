## 문제 설명

알파벳 문자열 `문장`에서 **대문자 개수**와 **소문자 개수**를 세어 차례로 말하기로 출력하세요.

알파벳 외 문자(공백·숫자·특수문자)는 **모두 무시**합니다.

### 입력
- `문장`: 알파벳·공백·숫자·특수문자 혼합 문자열 (채점 시 자동 설정)

### 출력
1. 대문자 개수를 말하기
2. 소문자 개수를 말하기

### 예시
`문장 = "HelloWorld"` → 출력: `2`, `8`
`문장 = "HELLO"` → 출력: `5`, `0`
`문장 = "abc123"` → 출력: `0`, `3`

### 힌트
- `계산`의 `X번째 글자` 블록으로 한 글자씩 꺼내기
- 영문 대문자·소문자의 유니코드 범위 비교 (대문자: `A`~`Z`, 소문자: `a`~`z`)
- 대문자 카운터·소문자 카운터·현재 글자 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 두 카운터(대문자·소문자)를 `0`으로 초기화
- `i = 1..문자열 길이` 반복:
  - 현재 = `i번째 글자`
  - 현재가 `"A"`와 `"Z"` 사이면 대문자 +1
  - 현재가 `"a"`와 `"z"` 사이면 소문자 +1
- 반복 후 두 값을 차례로 말하기

</details>

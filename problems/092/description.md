## 문제 설명

리스트 `숫자들`에서 **2번째로 큰 값**을 찾아 말하기로 출력하세요. 정렬했을 때 두 번째 위치에 오는 값 (중복 허용).

- `[5, 3, 2]` → `3`
- `[5, 5, 3]` → `5` (같은 값 두 번이어도 두 번째 자리에 `5`)

### 입력
- `숫자들`: 2개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 2번째로 큰 값 말하기

### 예시
`[5, 3, 2]` → 출력: `3`
`[10, 20, 30]` → 출력: `20`
`[7, 7, 7]` → 출력: `7`

### 힌트
- 최댓값과 두번째값 두 변수를 각각 추적
- 새 항목이 최댓값보다 크면 최댓값은 두번째값으로 내려가고, 새 항목이 최댓값이 됨
- `판단`의 `>`, `≥` 블록 활용

<details>
<summary>자세한 접근법</summary>

- 두 변수를 리스트의 첫 항목으로 초기화 (최댓값·두번째값 모두 `숫자들[1번째]`)
- `i = 2..숫자들 항목 수` 반복하며 현재 값을 검사:
  - 현재 > 최댓값 → 두번째값 = 최댓값, 최댓값 = 현재
  - 현재 ≤ 최댓값 **그리고** 현재 > 두번째값 → 두번째값 = 현재
  - **주의**: 중복 허용이므로 `현재 = 최댓값`이어도 두번째값 갱신 가능하게 `≤`로 비교
- 반복 종료 후 두번째값 말하기

</details>

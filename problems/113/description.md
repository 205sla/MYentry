## 문제 설명

자연수 `N`을 **연속된 자연수의 합**으로 나타낼 수 있는 방법의 **가짓수**를 구해 말하기로 출력하세요. `N` 자체(길이 1)도 한 가지로 셉니다.

예: `N = 15`
- `15`
- `7 + 8`
- `4 + 5 + 6`
- `1 + 2 + 3 + 4 + 5`

→ `4가지`

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- 연속 자연수 합 표현 가짓수를 말하기

### 예시
`N = 1` → 출력: `1`
`N = 9` → 출력: `3` (9, 4+5, 2+3+4)
`N = 15` → 출력: `4`

### 힌트
- **투 포인터**: 왼쪽·오른쪽 포인터가 모두 `1`부터 시작, 합을 유지하며 이동
  - 합 < N → 오른쪽 +1 (합에 오른쪽 값 더하기)
  - 합 = N → 가짓수 +1, 왼쪽을 빼고 오른쪽으로 밀기
  - 합 > N → 왼쪽 값 빼고 왼쪽 +1
- 왼쪽이 `N`을 넘으면 종료
- 포인터·합·개수 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 왼쪽 = 1, 오른쪽 = 1, 현재합 = 1, 개수 = 0
- 왼쪽 ≤ N 인 동안 반복:
  - 현재합 = N → 개수 += 1. 현재합에서 왼쪽 값 빼고 왼쪽 += 1
  - 현재합 < N → 오른쪽 += 1, 현재합에 오른쪽 값 더하기
  - 현재합 > N → 현재합에서 왼쪽 값 빼고 왼쪽 += 1
- 반복 종료 후 개수 말하기

</details>

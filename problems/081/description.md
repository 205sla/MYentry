## 문제 설명

별(`*`)을 거꾸로 줄여가는 직각삼각형을 `N`줄 **1초간씩** 차례로 말하기로 출력하세요. `1`번째 줄에는 `N`개, 마지막 줄에는 `1`개의 별.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
1. `*` × N개
2. `*` × (N-1)개
3. …
N. `*` (1개)

모든 말하기는 **1초간**.

### 예시
`N = 3` →
- 1초간 `***`
- 1초간 `**`
- 1초간 `*`

### 힌트
- 바깥 반복 `i = 1..N`에서 현재 줄의 별 개수는 `(N - i + 1)`
- 또는 카운터를 `N`부터 `1`까지 줄이며 반복
- 줄 문자열과 카운터 변수는 직접 만들어 쓰세요

## 문제 설명

리스트 `숫자들`에서 **홀수만** 추출해 원래 순서대로 리스트 `홀수들`에 담으세요.

### 입력
- `숫자들`: 정수 리스트 (채점 시 자동 설정)
- `홀수들`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `홀수들`: `숫자들`의 홀수 원소들 (원래 순서 유지)

### 예시
`숫자들 = [1, 2, 3, 4, 5]` → `홀수들 = [1, 3, 5]`
`숫자들 = [2, 4, 6]` → `홀수들 = []`
`숫자들 = [-3, -2, 0, 1]` → `홀수들 = [-3, 1]`

### 힌트
- `흐름`의 `N번 반복` 블록으로 리스트 순회
- `계산`의 `나누기의 나머지` 블록으로 홀짝 판정
- `자료`의 `항목 추가하기` 블록

## 문제 설명

영어 **소문자**만으로 이루어진 문자열 `문장`의 각 글자를 `N`칸 뒤로 이동시킨 **시저 암호**를 말하기로 출력하세요. `z`를 넘으면 다시 `a`부터 순환합니다.

예: `N = 3`
- `a` → `d`, `b` → `e`, …, `x` → `a`, `y` → `b`, `z` → `c`

### 입력
- `문장`: 영어 소문자만 포함된 문자열 (채점 시 자동 설정)
- `N`: 0 이상 25 이하의 정수 (채점 시 자동 설정)

### 출력
- 암호화된 문자열 말하기

### 예시
`문장 = "abc", N = 3` → 출력: `def`
`문장 = "xyz", N = 3` → 출력: `abc`
`문장 = "hello", N = 1` → 출력: `ifmmp`
`문장 = "anything", N = 0` → 출력: `anything`

### 힌트
- 알파벳 26자를 담은 문자열 변수를 직접 만들어 참조 테이블로 쓰기
- 각 글자의 위치(0~25)를 찾은 뒤 `(위치 + N) mod 26` 로 암호화
- 결과·위치 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 알파벳 변수에 `"abcdefghijklmnopqrstuvwxyz"` 저장
- 결과를 `""`로 초기화
- 문장의 각 글자에 대해:
  - 알파벳에서 그 글자의 위치 `p`를 찾음 (0-based)
  - 새 위치 = `(p + N) mod 26`
  - 결과 += 알파벳의 (새 위치 + 1)번째 글자
- 반복 후 결과 말하기

## 문제 설명

두 변수 `변수1`과 `변수2`의 **차**(`변수1 - 변수2`)를 말하기로 출력하세요.

### 입력
- `변수1`: 첫 번째 정수 (채점 시 자동 설정)
- `변수2`: 두 번째 정수 (채점 시 자동 설정)

### 출력
- `변수1 - 변수2` 결과를 말하기

### 예시
변수1: `10`, 변수2: `3` → 출력: `7`
변수1: `5`, 변수2: `8` → 출력: `-3`

### 힌트
- `계산` 카테고리의 뺄셈 블록과 `자료` 카테고리의 변수값 블록을 조합
- `생김새` 카테고리의 `말하기` 블록으로 결과 출력

## 문제 설명

0 이상의 정수 `N`을 **2진수 문자열**로 변환해 말하기로 출력하세요.

### 입력
- `N`: 0 이상의 정수 (채점 시 자동 설정)

### 출력
- `N`의 2진수 표현을 말하기 (문자열)

### 예시
입력: `N = 0` → 출력: `"0"`
입력: `N = 5` → 출력: `"101"`
입력: `N = 13` → 출력: `"1101"`
입력: `N = 255` → 출력: `"11111111"`

### 힌트
- `계산` 카테고리의 `나누기의 나머지` + `나누기의 몫`, `합치기` 블록
- `흐름`의 `...인 동안 반복`으로 `N`이 `0`보다 큰 동안 반복
- 나머지와 결과 문자열을 담을 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 결과 문자열 변수를 `""`(빈 문자열)로 초기화
- `N`이 `0`보다 큰 동안 반복:
  - `N`을 `2`로 나눈 나머지를 구함
  - 결과 = (나머지)와 (결과) 합치기 — **나머지가 앞**
  - `N = N ÷ 2`의 **몫만** 남김
- **특수 케이스**: `N = 0`으로 시작하면 반복이 한 번도 돌지 않으니 결과가 빈 문자열 → 이때는 `"0"`을 말하기
- **주의**: 결과 변수를 엔트리 기본값 `0`으로 두면 `0` 뒤에 숫자가 붙습니다. 반드시 빈 문자열로 초기화

</details>

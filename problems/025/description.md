## 문제 설명

0 이상의 정수 `N`을 **2진수 문자열**로 변환해 말하기로 출력하세요.

### 입력
- `N`: 0 이상의 정수 (채점 시 자동 설정)

### 출력
- `N`의 2진수 표현을 말하기 (문자열)

### 예시
입력: `N = 0` → 출력: `"0"`
입력: `N = 5` → 출력: `"101"`
입력: `N = 13` → 출력: `"1101"`
입력: `N = 255` → 출력: `"11111111"`

### 힌트
- 변수 `N`, `나머지`, `결과`는 **기본으로 제공**됩니다
- ⚠️ 변수 `결과`는 엔트리 기본값 `0`으로 시작합니다 — **반복을 시작하기 전에 반드시 `결과`를 빈 문자열(`""`)로 초기화**해야 합니다 (아니면 문자열 합치기에서 `0`이 맨 앞에 붙습니다)
- 핵심 아이디어: `N`을 2로 계속 나누면서 **나머지를 `결과`의 앞쪽에 붙여가기**
  - `나머지 = N을 2로 나눈 나머지`
  - `결과 = (나머지)와 (결과) 합치기` — 순서 주의 (나머지가 앞)
  - `N = N ÷ 2`의 **몫만** 남기기 (소수점 아래 버림)
  - `N = 0`이 될 때까지 반복
- **특수 케이스**: `N = 0`으로 시작하면 반복이 한 번도 돌지 않아 `결과`가 빈 문자열 → 이때는 `"0"`을 말하기
- `흐름` 카테고리의 `...인 동안 반복하기`, `계산` 카테고리의 `나누기의 몫` / `나누기의 나머지`, `계산`의 `합치기` 블록을 활용하세요

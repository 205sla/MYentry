## 문제 설명

구구단 **2단**을 9줄 차례로 **1초간씩** 말하기로 출력하세요.

### 입력
- 없음

### 출력
1. `2 x 1 = 2`를 **1초간** 말하기
2. `2 x 2 = 4`를 **1초간** 말하기
3. `2 x 3 = 6`를 **1초간** 말하기
4. `2 x 4 = 8`를 **1초간** 말하기
5. `2 x 5 = 10`를 **1초간** 말하기
6. `2 x 6 = 12`를 **1초간** 말하기
7. `2 x 7 = 14`를 **1초간** 말하기
8. `2 x 8 = 16`를 **1초간** 말하기
9. `2 x 9 = 18`를 **1초간** 말하기

### 힌트
- `생김새` 카테고리의 `X초 동안 말하기` 블록을 9번 이어서 쓰기
- 각 결과값은 직접 계산해서 문자열로 써주세요 (예: `2 x 3 = 6`)

## 문제 설명

두 리스트 `A`와 `B`의 **교집합**을 리스트 `교집합`에 담으세요. `A`에 먼저 등장한 순서대로, 중복 없이 담습니다.

### 입력
- `A`, `B`: 정수 리스트 (채점 시 자동 설정)
- `교집합`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `교집합`: `A`와 `B`에 모두 포함된 값들 (A 순서, 중복 제거)

### 예시
`A = [1, 2, 3]`, `B = [2, 3, 4]` → `교집합 = [2, 3]`
`A = [1, 1, 2, 2]`, `B = [1, 2]` → `교집합 = [1, 2]`
`A = [1, 2]`, `B = [3, 4]` → `교집합 = []`

### 힌트
- `A`의 각 항목에 대해 두 가지 확인:
  1. 그 항목이 `B`에도 있는가?
  2. 이미 `교집합`에 추가한 적이 없는가?
- 둘 다 만족하면 `교집합`에 추가
- 판정 플래그 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- `A`를 순회하며 현재 항목 `값`에 대해
  - `B에있음 = 0`, `B`를 순회하며 `값`과 같은 항목 있으면 `1`
  - `이미있음 = 0`, `교집합`을 순회하며 `값`과 같은 항목 있으면 `1`
  - 두 조건(B에있음 = 1 그리고 이미있음 = 0) 만족하면 `교집합`에 `값` 추가

</details>

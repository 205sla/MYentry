## 문제 설명

`1`부터 `N`까지의 **합**을 구해 말하기로 출력하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `1 + 2 + ... + N` 결과를 말하기

### 예시
`N = 1` → 출력: `1`
`N = 5` → 출력: `15`
`N = 10` → 출력: `55`

### 힌트
- `흐름`의 `N번 반복하기` 블록 또는 `...인 동안 반복`
- 누적 합을 저장할 보조 변수를 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 합 변수를 `0`으로 초기화
- 카운터 변수를 `1`로 시작해 `N`까지 반복하며 합에 카운터를 더하기
- 또는 반복 블록 안에서 `i`를 이용해 1~N을 차례로 더하기

</details>

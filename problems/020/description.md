## 문제 설명

`리스트`에 들어 있는 숫자들의 **최댓값**, **최솟값**, **평균**을 각각 구해 말하기로 출력하세요.

### 입력
- `리스트`: 1개 이상의 숫자 (채점 시 자동 설정)

### 출력
- 최댓값을 말하기
- 최솟값을 말하기
- 평균을 말하기

### 예시
입력: `[1, 2, 3]` → 최댓값 `3`, 최솟값 `1`, 평균 `2`
입력: `[10, 20, 30, 40, 50]` → 최댓값 `50`, 최솟값 `10`, 평균 `30`

### 힌트
- 리스트 `리스트`와 변수 `최댓값`, `최솟값`, `평균`은 **기본으로 제공**됩니다
- 처음에는 `최댓값`과 `최솟값`을 `리스트의 1번째 항목`으로 초기화하세요
- `흐름` 카테고리의 `N번 반복하기`와 `자료` 카테고리의 `리스트 항목 수`로 순회하세요
- `판단` 카테고리의 `>`, `<` 블록으로 현재 값과 비교해서 갱신하세요
- 평균은 `평균` 변수에 값을 누적한 뒤 `리스트 항목 수`로 나누면 됩니다

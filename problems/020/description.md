## 문제 설명

`리스트`에 들어 있는 숫자들의 **최댓값**, **최솟값**, **평균**을 각각 구해 말하기로 출력하세요.

### 입력
- `리스트`: 1개 이상의 숫자 (채점 시 자동 설정)

### 출력
- 최댓값을 말하기
- 최솟값을 말하기
- 평균을 말하기

### 예시
입력: `[1, 2, 3]` → 최댓값 `3`, 최솟값 `1`, 평균 `2`
입력: `[10, 20, 30, 40, 50]` → 최댓값 `50`, 최솟값 `10`, 평균 `30`

### 힌트
- `자료` 카테고리의 `N번째 항목`으로 하나씩 꺼내며 비교
- `판단`의 `>`, `<` 블록으로 현재값과 비교해 갱신
- 세 결과를 담을 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 최댓값·최솟값을 리스트의 `1번째 항목`으로 초기화
- 합 누적 변수를 `0`으로 초기화
- 리스트 길이만큼 반복하며:
  - 현재 항목이 최댓값보다 크면 갱신
  - 현재 항목이 최솟값보다 작으면 갱신
  - 합에 현재 항목을 더하기
- 반복 종료 후 평균 = 합 ÷ 리스트 항목 수
- 세 값을 차례로 말하기

</details>

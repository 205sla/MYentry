## 문제 설명

문자열 `문장`을 공백(`" "`) 기준으로 나눴을 때 **가장 긴 단어**를 말하기로 출력하세요. 길이가 같은 단어가 여럿이면 **먼저 나온 단어**를 출력합니다.

### 입력
- `문장`: 적어도 한 단어 이상을 포함한 문자열 (채점 시 자동 설정)

### 출력
- 가장 긴 단어 말하기

### 예시
`"hello world"` → `hello` (동률 → 먼저)
`"a bb ccc"` → `ccc`
`"the quick brown fox"` → `quick`

### 힌트
- 현재까지 본 가장 긴 단어와 지금 쌓고 있는 단어를 각각 관리
- 공백을 만나거나 문장이 끝나면 쌓은 단어의 길이를 최장과 비교
- 최장·현재단어 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 최장을 `""`, 현재단어를 `""` 로 초기화
- 문장을 한 글자씩 순회:
  - 공백 만나면 **현재단어의 길이 > 최장의 길이** 일 때만 최장 = 현재단어. 현재단어 초기화
  - 아니면 현재단어에 글자 이어붙이기
- 반복 종료 후 마지막 단어도 같은 비교 한 번 더
- 최장 말하기

</details>

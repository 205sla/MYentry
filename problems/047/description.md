## 문제 설명

두 변수 `변수1`을 `변수2`로 나눈 **몫**(정수 나누기)을 말하기로 출력하세요.

### 입력
- `변수1`: 나뉨수 정수 (채점 시 자동 설정)
- `변수2`: 나눔수 정수, `0`이 아님 (채점 시 자동 설정)

### 출력
- `변수1 ÷ 변수2`의 몫을 말하기 (소수점 아래 버림)

### 예시
변수1: `10`, 변수2: `3` → 출력: `3`
변수1: `20`, 변수2: `4` → 출력: `5`

### 힌트
- `계산` 카테고리의 `나누기의 몫` 블록 사용
- `자료` 카테고리의 변수값 블록과 조합
- `생김새` 카테고리의 `말하기` 블록으로 출력

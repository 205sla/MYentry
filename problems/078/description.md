## 문제 설명

변수 `문장`에서 특정 글자 `찾을값`이 **몇 번 나오는지** 세어 말하기로 출력하세요.

### 입력
- `문장`: 임의의 문자열 (채점 시 자동 설정)
- `찾을값`: 한 글자 (채점 시 자동 설정)

### 출력
- `문장`에서 `찾을값`이 나온 횟수를 말하기

### 예시
`문장 = "banana"`, `찾을값 = "a"` → 출력: `3`
`문장 = "hello world"`, `찾을값 = "o"` → 출력: `2`
`문장 = "abc"`, `찾을값 = "z"` → 출력: `0`

### 힌트
- `흐름`의 `N번 반복` 블록으로 `1..문자열 길이` 순회
- `계산`의 `X번째 글자` + `판단`의 `=` 블록으로 비교
- 카운터 변수는 직접 만들어 쓰세요

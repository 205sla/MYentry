## 문제 설명

1 이상의 정수 `N`이 몇 자리 수인지 세어 말하기로 출력하세요.

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `N`의 자릿수

### 예시
`N = 7` → 출력: `1`
`N = 42` → 출력: `2`
`N = 1000` → 출력: `4`

### 힌트
- `10으로 나눈 몫` 블록으로 자릿수를 한 자리씩 떨어뜨리며 세기
- 카운터 변수와 임시값 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 카운터를 `0`, 남은 = `N`
- 남은 > 0 인 동안:
  - 남은 = 남은 ÷ 10 (몫)
  - 카운터 += 1
- 반복 끝난 뒤 카운터 말하기

</details>

## 문제 설명

변수 `문장`이 **회문(palindrome)** 인지 판별해 말하기로 출력하세요. 회문은 앞에서 읽어도 뒤에서 읽어도 같은 문자열입니다.

- 회문이면 `회문`
- 아니면 `회문 아님`
- 한 글자와 빈 문자열은 회문으로 간주

### 입력
- `문장`: 임의의 문자열 (채점 시 자동 설정)

### 출력
- `회문` 또는 `회문 아님` 말하기

### 예시
`문장 = "aba"` → 출력: `회문`
`문장 = "hello"` → 출력: `회문 아님`
`문장 = "기러기"` → 출력: `회문`

### 힌트
- `계산`의 `X번째 글자`, `문자열 길이`, `판단`의 `=` 블록
- 양쪽 끝에서 안쪽으로 좁혀가며 비교
- 판정 플래그·포인터 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 회문 플래그를 `1`(회문)으로 초기화
- `i = 1..(길이 ÷ 2)` 반복:
  - `(i)번째 글자 ≠ (길이 - i + 1)번째 글자` 이면 플래그 `0`
- 반복 후 플래그가 `1` 이면 `회문`, 아니면 `회문 아님` 말하기

</details>

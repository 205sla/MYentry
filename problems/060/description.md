## 문제 설명

`1`부터 `N`까지의 **곱**(팩토리얼 `N!`)을 구해 말하기로 출력하세요.

### 입력
- `N`: 0 이상의 정수 (채점 시 자동 설정, 0~12 범위)
- 참고: `0! = 1` (수학적 정의)

### 출력
- `N!` 값을 말하기

### 예시
`N = 0` → 출력: `1`
`N = 1` → 출력: `1`
`N = 5` → 출력: `120`
`N = 7` → 출력: `5040`

### 힌트
- 곱을 누적할 보조 변수는 직접 만들어 쓰세요 (초기값 `1`)
- `흐름`의 `N번 반복` 블록
- `계산`의 곱셈 블록으로 `누적값 × 카운터`를 저장

<details>
<summary>자세한 접근법</summary>

- 결과 변수를 `1`로 초기화 (곱셈 누적에서 `0`이면 항상 0이 되므로 주의)
- 카운터 `i = 1..N` 반복하며 결과 = 결과 × `i`
- `N = 0` 이면 반복을 한 번도 돌지 않아 결과는 `1` 그대로

</details>

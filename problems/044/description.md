## 문제 설명

`묻고 기다리기`로 이름을 입력받아 `안녕 OOO님` 형식으로 말하세요.

### 입력
- `묻고 기다리기` 블록으로 이름 문자열을 입력받습니다 (채점 시 자동 응답)

### 출력
- `안녕 (이름)님` 말하기 (예: 이름이 `엔트리`면 `안녕 엔트리님`)

### 예시
입력: `엔트리` → 출력: `안녕 엔트리님`
입력: `지민` → 출력: `안녕 지민님`

### 힌트
- `자료` 카테고리의 `묻고 기다리기` 블록
- `계산` 카테고리의 `합치기` 블록을 두 번 중첩해 `"안녕 " + 대답 + "님"` 만들기
- `생김새` 카테고리의 `말하기` 블록

<details>
<summary>자세한 접근법</summary>

- 묻고 기다리기 블록을 실행하면 입력값이 `대답`에 저장됨
- 합치기 블록으로 `안녕 ` / `대답` / `님` 세 조각을 하나의 문자열로 연결
- 말하기 블록으로 출력

</details>

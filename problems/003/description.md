## 문제 설명

`리스트`에 들어 있는 숫자들을 **오름차순**으로 정렬하세요.

### 입력
- `리스트`: 정렬할 숫자들 (채점 시 자동 설정)

### 출력
- `리스트`의 내용이 오름차순으로 정렬된 상태

### 예시
입력: `[3, 1, 2]` → 리스트 최종 상태: `[1, 2, 3]`
입력: `[5, 2, 8, 1]` → 리스트 최종 상태: `[1, 2, 5, 8]`

### 힌트
- 버블 정렬, 삽입 정렬 등 자유롭게 구현 가능합니다
- `자료` 카테고리의 리스트 관련 블록을 활용하세요
- 미리 만들어진 변수 `임시값`, `현위치`, `반복끝값`, `반복시작값`을 활용하세요

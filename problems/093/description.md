## 문제 설명

리스트 `숫자들`에서 **K번째로 큰 값**을 찾아 말하기로 출력하세요. 내림차순으로 정렬했을 때 `K`번째에 오는 값 (중복 허용).

- `[10, 20, 30, 40, 50]`의 K=2 → `40`
- `[5, 5, 3]`의 K=2 → `5` (정렬 [5,5,3]의 두 번째)

### 입력
- `숫자들`: K개 이상의 정수 리스트 (채점 시 자동 설정)
- `K`: 1 이상 리스트 길이 이하의 자연수 (채점 시 자동 설정)

### 출력
- K번째로 큰 값 말하기

### 예시
`숫자들=[10,20,30,40,50], K=2` → 출력: `40`
`숫자들=[3,1,4,1,5,9,2,6], K=3` → 출력: `5`
`숫자들=[5,5,3], K=2` → 출력: `5`

### 힌트
- 보조 리스트에 `숫자들`을 복사
- K-1번 반복하며 보조 리스트에서 최댓값을 찾아 **삭제**
- 남은 보조 리스트의 최댓값이 답

  - 보조 리스트 전체를 훑어 최댓값 위치를 찾고 그 항목 삭제
- 반복 종료 후 보조 리스트의 최댓값을 다시 찾아 말하기

## 문제 설명

문자열 `문장`에서 연속된 같은 글자를 `(글자)(개수)` 형식으로 압축해 말하기로 출력하세요.

- 1번만 등장해도 `글자1`로 표기
- 빈 문자열 입력은 주어지지 않음

### 입력
- `문장`: 비어있지 않은 문자열 (채점 시 자동 설정)

### 출력
- 압축된 문자열 말하기

### 예시
`"aaabb"` → `a3b2`
`"abcabc"` → `a1b1c1a1b1c1`
`"aaaa"` → `a4`
`"x"` → `x1`

### 힌트
- 앞에서부터 훑으며 현재 글자와 다음 글자 비교
- 같으면 현재 구간 카운트 +1, 다르면 `(글자)(카운트)`를 결과에 붙이고 새로 시작
- 결과 문자열·현재 글자·카운트 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 결과를 `""`, 현재 글자를 문자열의 1번째 글자, 카운트를 `1`로 초기화
- `i = 2..문자열 길이` 반복:
  - 현재 글자 = `i번째 글자` 이면 카운트 +1
  - 다르면 결과 += (현재 글자 + 카운트), 현재 글자 = `i번째 글자`, 카운트 = `1`
- 반복 종료 후 마지막 구간을 결과에 추가하고 말하기

</details>

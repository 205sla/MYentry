## 문제 설명

`2` 이상 `N` 이하의 **소수 개수**를 세어 말하기로 출력하세요.

소수는 1과 자기 자신만을 약수로 갖는 1보다 큰 자연수입니다.

### 입력
- `N`: 2 이상의 자연수 (채점 시 자동 설정)

### 출력
- N 이하 소수 개수를 말하기

### 예시
`N = 10` → 출력: `4` (2, 3, 5, 7)
`N = 20` → 출력: `8`
`N = 2` → 출력: `1`

### 힌트
- 바깥 반복 `i = 2..N`에서 각 `i`가 소수인지 검사
- 소수 판정: `2..(i-1)` 중 `i`를 나누어떨어지게 하는 수가 있는지
- 소수 카운터·약수 플래그 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 카운터를 `0`으로 초기화
- `i = 2..N` 반복
  - 약수 플래그를 `0`으로 초기화
  - `j = 2..(i-1)` 반복: `i mod j = 0` 이면 약수 플래그 `1`
  - 약수 플래그가 `0`이면 `i`는 소수 → 카운터 +1
- 반복 끝난 뒤 카운터 말하기
- (속도 개선: `j`를 `√i` 까지만 검사해도 됩니다)

</details>

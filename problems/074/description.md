## 문제 설명

0 이상의 정수 `N`의 **각 자릿수의 합**을 구해 말하기로 출력하세요.

### 입력
- `N`: 0 이상의 정수 (채점 시 자동 설정)

### 출력
- 각 자리 숫자를 모두 더한 값을 말하기

### 예시
`N = 0` → 출력: `0`
`N = 7` → 출력: `7`
`N = 123` → 출력: `6` (1+2+3)
`N = 999` → 출력: `27`

### 힌트
- `10으로 나눈 나머지` 블록으로 일의 자리 얻기
- `10으로 나눈 몫` 블록으로 자릿수 하나씩 떨어내기
- 남은 값이 `0`이 될 때까지 반복
- 합·남은 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 합 변수를 `0`, 남은 = `N`
- 남은 > 0 인 동안 반복:
  - 합 = 합 + (남은 mod 10)
  - 남은 = 남은 ÷ 10 (몫)
- `N = 0`이면 반복을 돌지 않아 합이 그대로 `0`

</details>

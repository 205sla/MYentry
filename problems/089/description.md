## 문제 설명

두 자연수 `A`와 `B`의 **최소공배수(LCM)** 을 구해 말하기로 출력하세요.

LCM은 `A`와 `B`의 공통된 배수 중 가장 작은 자연수입니다.

### 입력
- `A`, `B`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- `A`와 `B`의 최소공배수를 말하기

### 예시
`A=4, B=6` → 출력: `12`
`A=3, B=5` → 출력: `15`
`A=7, B=7` → 출력: `7`

### 힌트
- 공식: `LCM(A, B) = A × B ÷ GCD(A, B)`
- GCD는 유클리드 호제법 또는 단순 탐색으로 구할 수 있음
- GCD·임시값 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 방법 1: 유클리드 호제법으로 GCD 구한 뒤 `A × B ÷ GCD`
  - `x = A`, `y = B`
  - `y ≠ 0` 동안: 임시 = `x mod y`, `x = y`, `y = 임시`
  - 반복 종료 후 `x`가 GCD
- 방법 2: `max(A, B)`부터 시작해서 `A`와 `B`로 둘 다 나누어떨어지는 배수를 찾을 때까지 증가

</details>

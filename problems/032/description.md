## 문제 설명

리스트 `숫자들`에서 **엄격히 증가하는 부분수열(LIS: Longest Increasing Subsequence)**의 **길이**를 구해 말하기로 출력하세요.

- "부분수열"은 연속되지 않아도 되며 원래 순서는 지킵니다 (일부 항목 건너뛰기 가능)
- "엄격히 증가"는 앞 항목 < 뒤 항목 (같거나 작으면 안 됨)

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- LIS의 길이를 말하기

### 예시
입력: `[3,1,4,1,5,9,2,6]` → 출력: `4` (예: `[1,4,5,9]`)
입력: `[1,2,3,4,5]` → 출력: `5` (전체가 LIS)
입력: `[5,4,3,2,1]` → 출력: `1` (모두 내림차순)

### 힌트
- 각 위치에서 "그 위치에서 끝나는 LIS 길이"를 저장할 보조 리스트를 직접 만드세요 (O(N²) DP)
- `흐름`의 `N번 반복`을 **이중 중첩**
- `판단`의 `그리고`·`<`·`>` 블록 활용

<details>
<summary>자세한 접근법</summary>

- **1단계 (초기화)**: 길이 리스트를 비우고 `숫자들의 항목 수`만큼 `1`을 추가 (자기 자신 하나만 고르면 길이 1)
- **2단계 (DP)**: 바깥 반복 `i = 2..숫자들의 항목 수`
  - 안쪽 반복 `j = 1..i-1`:
    - `숫자들의 j번째 < 숫자들의 i번째` **그리고** `길이의 j번째 + 1 > 길이의 i번째` 이면
      - 길이의 i번째 항목 = 길이의 j번째 + 1
- **3단계 (최댓값)**: 길이 리스트의 최댓값을 말하기

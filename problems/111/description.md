## 문제 설명

수직선 위 위치 `N`에서 출발해 다음 세 가지 연산으로 `K`에 도달하는 **최소 연산 횟수**를 구해 말하기로 출력하세요.

- `+1`: 위치가 `1` 증가
- `-1`: 위치가 `1` 감소
- `×2`: 위치가 두 배

- `0 ≤ N, K ≤ 100` 범위 안에서만 이동 (`-1 < 0`인 지점은 진입 불가)

### 입력
- `N`: 시작 위치 (0 이상 100 이하)
- `K`: 도착 위치 (0 이상 100 이하)
(채점 시 자동 설정)

### 출력
- 최소 연산 횟수를 말하기

### 예시
`N = 5, K = 17` → 출력: `4` (5→10→9→18→17)
`N = 5, K = 5` → 출력: `0`
`N = 0, K = 1` → 출력: `1`
`N = 1, K = 4` → 출력: `2` (1→2→4)

### 힌트
- **BFS**: 위치별 최단 거리를 구하는 표준 탐색
- 큐 역할을 할 1차원 리스트 두 개(위치 큐·거리 큐)와 방문 체크용 리스트 1개를 직접 만드세요
- 한 번에 세 방향으로 확장: `현재+1`, `현재-1`, `현재×2`

<summary>자세한 접근법</summary>

- 방문 리스트를 `101`칸(또는 여유있게 더 큰 크기)의 `0`으로 초기화
- 위치 큐·거리 큐에 `N`과 `0`을 넣고 방문 표시
- 큐가 빌 때까지:
  - 맨 앞 위치·거리를 꺼냄 (1번째 항목 삭제)
  - 위치가 `K` 이면 그 거리가 답
  - 아니면 세 이웃(`+1`, `-1`, `*2`)에 대해:
    - 범위 안이고 미방문이면 방문 표시 후 큐에 추가 (거리+1)

## 문제 설명

리스트 `숫자들`의 모든 항목을 **뒤에서부터** 차례로 **1초간씩** 말하기로 출력하세요.

### 입력
- `숫자들`: 1개 이상의 항목이 있는 리스트 (채점 시 자동 설정)

### 출력
1. 리스트의 **마지막** 항목을 **1초간** 말하기
2. 그 이전 항목을 **1초간** 말하기
3. … 계속해서 1번째 항목까지

### 예시
`숫자들 = [1, 2, 3]` → `3`, `2`, `1`
`숫자들 = [10, 20, 30, 40]` → `40`, `30`, `20`, `10`

### 힌트
- `흐름`의 `N번 반복` 블록
- `자료`의 `N번째 항목` + `항목 수` 블록
- 위치를 가리키는 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 위치 변수를 `리스트의 항목 수`로 초기화
- `항목 수`번 반복하며 `위치번째 항목`을 1초간 말한 뒤 위치 `-1`
- 또는 `i = 1..항목 수` 반복에서 `(항목 수 - i + 1)번째 항목`을 말하기

</details>

## 문제 설명

자연수 `N`을 **소인수분해**한 결과를 리스트 `인수들`에 **오름차순**으로 담으세요. 중복되는 소인수는 등장 횟수만큼 반복해 넣습니다.

- `12 = 2 × 2 × 3` → `[2, 2, 3]`
- `60 = 2² × 3 × 5` → `[2, 2, 3, 5]`
- `17`은 소수 → `[17]`
- `1`은 소인수 없음 → `[]`

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- 리스트 `인수들`: `N`의 소인수들 (오름차순, 중복 포함)

### 예시
입력: `N = 12` → `인수들 = [2, 2, 3]`
입력: `N = 100` → `인수들 = [2, 2, 5, 5]`
입력: `N = 7` → `인수들 = [7]`

### 힌트
- `흐름` 카테고리의 `...인 동안 반복` 블록을 활용
- 작은 소인수부터 차례로 나눠가며 `인수들`에 쌓기
- 나누는 후보 값을 담을 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 후보 변수를 `2`로 시작
- `N`이 후보로 나누어떨어지는 **동안** 반복: `인수들`에 후보를 추가하고 `N`을 후보로 나누기
- 더 이상 나누어떨어지지 않으면 후보를 `1` 증가
- `N`이 `1`이 될 때까지 바깥쪽 반복을 계속
- 반복문 두 겹 중첩이 필요 (바깥: `N > 1`, 안쪽: `N`이 후보로 나누어떨어지는 동안)

</details>

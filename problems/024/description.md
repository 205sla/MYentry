## 문제 설명

자연수 `N`을 **소인수분해** 한 결과를 리스트 `인수들`에 **오름차순**으로 채워 넣으세요. 중복되는 소인수는 등장한 횟수만큼 반복해 넣습니다.

- `12 = 2 × 2 × 3` → `인수들 = [2, 2, 3]`
- `60 = 2 × 2 × 3 × 5` → `인수들 = [2, 2, 3, 5]`
- `17`은 소수 → `인수들 = [17]`
- `1`은 소인수 없음 → `인수들 = []`

### 입력
- `N`: 1 이상의 자연수 (채점 시 자동 설정)

### 출력
- 리스트 `인수들`: `N`의 소인수들 (오름차순, 중복 포함)

### 예시
입력: `N = 12` → `인수들 = [2, 2, 3]`
입력: `N = 100` → `인수들 = [2, 2, 5, 5]`
입력: `N = 7` → `인수들 = [7]`

### 힌트
- 변수 `N`, `i`는 **기본으로 제공**됩니다
- 리스트 `인수들`도 **기본으로 제공**됩니다 (처음엔 비어 있음)
- 핵심: 2부터 시작해 나누어떨어지는 동안 계속 나누기
  - `i`를 `2`부터 시작
  - `N`이 `i`로 나누어떨어지는 **동안** 반복: `인수들`에 `i`를 추가하고 `N = N ÷ i`
  - 더 이상 나누어떨어지지 않으면 `i`를 1 증가
  - `N`이 `1`이 될 때까지 반복
- `흐름` 카테고리의 `...인 동안 반복하기` 블록을 두 겹으로 중첩 (바깥: `N > 1`, 안쪽: `N을 i로 나눈 나머지 = 0`)
- `자료` 카테고리의 `항목 추가하기` 블록으로 소인수를 `인수들`에 쌓으세요

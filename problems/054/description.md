## 문제 설명

두 변수 `A`와 `B`의 대소를 비교해 말하기로 출력하세요.

### 입력
- `A`, `B`: 정수 (채점 시 자동 설정)

### 출력
- `A > B` 이면 `A가 큼`
- `A < B` 이면 `B가 큼`
- `A = B` 이면 `같음`

### 예시
`A = 5`, `B = 3` → 출력: `A가 큼`
`A = 2`, `B = 7` → 출력: `B가 큼`
`A = 4`, `B = 4` → 출력: `같음`

### 힌트
- `흐름` 카테고리의 `만약 ... 이라면, 아니면` 블록을 중첩
- `판단` 카테고리의 `>`, `<`, `=` 블록으로 비교

<details>
<summary>자세한 접근법</summary>

- `A > B` 이면 `A가 큼`
- 아니고 `A < B` 이면 `B가 큼`
- 아니면 `같음`

</details>

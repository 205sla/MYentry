## 문제 설명

연도 `Y`와 월 `M`이 주어질 때, 해당 달의 **일수**를 말하기로 출력하세요.

**윤년 규칙**:
- `4`로 나누어떨어지고 → 윤년 후보
- `100`으로 나누어떨어지면 평년 (윤년 취소)
- 하지만 `400`으로 나누어떨어지면 윤년 (규칙 복원)

**월별 일수**:
- 1·3·5·7·8·10·12월 → 31일
- 4·6·9·11월 → 30일
- 2월: 평년 28일 / 윤년 29일

### 입력
- `Y`: 연도 (예: 2024)
- `M`: 월 (1~12)
(채점 시 자동 설정)

### 출력
- 해당 달의 일수를 말하기

### 예시
`Y=2024, M=2` → 출력: `29` (2024는 윤년)
`Y=2023, M=2` → 출력: `28` (2023은 평년)
`Y=2000, M=2` → 출력: `29` (400의 배수 → 윤년)
`Y=1900, M=2` → 출력: `28` (100의 배수이지만 400 아님)
`Y=2024, M=7` → 출력: `31`

### 힌트
- 31일 달·30일 달·2월을 나눠서 처리
- 윤년 판정은 3단 조건문 (또는 `그리고`/`또는` 조합)
- 일수 결과 변수·윤년 플래그 변수는 직접 만들어 쓰세요

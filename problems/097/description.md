## 문제 설명

두 리스트 `A`와 `B`의 **차집합** `A - B`(A에 있지만 B에는 없는 값)를 리스트 `차집합`에 담으세요. `A`의 등장 순서대로, 중복 없이.

### 입력
- `A`, `B`: 정수 리스트 (채점 시 자동 설정)
- `차집합`: 빈 리스트 (채점 시 자동 설정)

### 출력
- `차집합`: `A`에는 있고 `B`에는 없는 값들 (A 순서, 중복 제거)

### 예시
`A = [1, 2, 3, 4]`, `B = [2, 4]` → `차집합 = [1, 3]`
`A = [1, 1, 2, 2]`, `B = [1]` → `차집합 = [2]`
`A = [1, 2]`, `B = [1, 2, 3]` → `차집합 = []`

### 힌트
- `A`를 순회하며 각 값에 대해 두 조건 확인:
  1. 그 값이 `B`에 **없는가**?
  2. 이미 `차집합`에 추가한 적 없는가?
- 둘 다 만족하면 `차집합`에 추가
- 판정 플래그 변수는 직접 만들어 쓰세요

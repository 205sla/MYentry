## 문제 설명

**오름차순으로 정렬된** 리스트 `숫자들`에서 `찾는값`의 위치를 **이진 탐색**으로 찾아 말하기로 출력하세요. 위치는 1번째 항목부터 셉니다.

- 값이 있으면: 해당 위치(1-based)를 말하기
- 값이 없으면: `-1`을 말하기

(테스트 케이스의 리스트에는 같은 값이 두 번 이상 나오지 않습니다.)

### 입력
- `숫자들`: **오름차순 정렬된** 정수 리스트 (채점 시 자동 설정)
- `찾는값`: 찾고자 하는 정수 (채점 시 자동 설정)

### 출력
- 찾은 위치(1-based) 또는 `-1`을 말하기

### 예시
입력: `숫자들 = [1,3,5,7,9]`, `찾는값 = 5` → 출력: `3`
입력: `숫자들 = [1,3,5,7,9]`, `찾는값 = 4` → 출력: `-1`
입력: `숫자들 = [10,20,30]`, `찾는값 = 30` → 출력: `3`

### 힌트
- `흐름` 카테고리의 `...인 동안 반복` 블록으로 탐색 범위를 반씩 줄이기
- `계산`의 `나누기의 몫`으로 중간 위치 계산
- 범위(시작·끝)와 중간 위치를 담을 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 범위 시작을 `1`, 끝을 `숫자들의 항목 수`로 초기화. 찾은 위치 변수는 `0`으로 초기화
- 시작이 끝보다 작거나 같고 아직 못 찾은 동안 반복:
  - 중간 = (시작 + 끝) ÷ 2의 몫
  - `숫자들의 중간번째 항목`이 `찾는값`과 같으면 찾음 위치에 중간을 저장 (반복 종료)
  - 해당 항목이 `찾는값`보다 작으면 시작을 중간 + 1로
  - 그 외는 끝을 중간 - 1로
- 반복이 끝나고 찾음 위치가 `0`이면 `-1` 말하기, 아니면 찾음 값 말하기

</details>

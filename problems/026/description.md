## 문제 설명

**오름차순으로 정렬된** 리스트 `숫자들`에서 `찾는값`의 위치를 **이진 탐색**으로 찾아 말하기로 출력하세요. 위치는 1번째 항목부터 셉니다.

- 값이 있으면: 해당 위치(1-based)를 말하기
- 값이 없으면: `-1`을 말하기

(테스트 케이스의 리스트에는 같은 값이 두 번 이상 나오지 않습니다.)

### 입력
- `숫자들`: **오름차순 정렬된** 정수 리스트 (채점 시 자동 설정)
- `찾는값`: 찾고자 하는 정수 (채점 시 자동 설정)

### 출력
- 찾은 위치(1-based) 또는 `-1`을 말하기

### 예시
입력: `숫자들 = [1,3,5,7,9]`, `찾는값 = 5` → 출력: `3`
입력: `숫자들 = [1,3,5,7,9]`, `찾는값 = 4` → 출력: `-1`
입력: `숫자들 = [10,20,30]`, `찾는값 = 30` → 출력: `3`

### 힌트
- 리스트 `숫자들`은 **기본으로 제공**됩니다 (채점 시 값이 교체됨)
- 변수 `찾는값`, `시작`, `끝`, `중간`, `찾음`은 **기본으로 제공**됩니다
- 핵심 아이디어: 탐색 범위를 매번 **절반**으로 줄여가기
  - 초기값: `시작 = 1`, `끝 = 숫자들의 항목 수`, `찾음 = 0`
  - `시작 <= 끝` **그리고** `찾음 = 0` 동안 반복:
    - `중간 = (시작 + 끝) ÷ 2`의 **몫** (정수 나눗셈)
    - 만약 `숫자들의 중간번째 항목 = 찾는값` → `찾음 = 중간`으로 저장 (반복 종료 조건)
    - 그렇지 않고 `숫자들의 중간번째 항목 < 찾는값` → `시작 = 중간 + 1`
    - 그렇지 않으면 → `끝 = 중간 - 1`
- 반복이 끝나면:
  - `찾음 = 0` 이면 `-1`을 말하기
  - 아니면 `찾음` 값을 말하기
- `흐름` 카테고리의 `...인 동안 반복하기`, `판단` 카테고리의 `그리고`·`=`·`<`, `계산`의 `나누기의 몫` 블록을 활용하세요

## 문제 설명

리스트 `숫자들`에서 **가장 자주 등장하는 값(최빈값)**을 찾아 말하기로 출력하세요. 동률이면 **더 작은 값**을 출력합니다.

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 최빈값을 말하기 (동률이면 더 작은 값)

### 예시
입력: `[1,2,2,3,3,3]` → 출력: `3` (3이 세 번)
입력: `[1,1,2,2]` → 출력: `1` (동률 → 작은 값)
입력: `[5]` → 출력: `5`

### 힌트
- 리스트 `숫자들`은 **기본으로 제공**됩니다 (채점 시 값이 교체됨)
- 변수 `최빈값`, `최대빈도`, `현위치`, `내부위치`, `값`, `빈도`는 **기본으로 제공**됩니다
- 핵심 아이디어: **이중 반복**으로 각 값의 등장 횟수를 세고 최댓값을 추적
  - 초기값: `최빈값 = 숫자들의 1번째 항목`, `최대빈도 = 0`
  - 바깥 반복 `현위치 = 1..숫자들의 항목 수`:
    - `값 = 숫자들의 현위치번째 항목`
    - `빈도 = 0`
    - 안쪽 반복 `내부위치 = 1..숫자들의 항목 수`:
      - 만약 `숫자들의 내부위치번째 항목 = 값` 이면 `빈도 = 빈도 + 1`
    - 만약 `빈도 > 최대빈도` → `최빈값 = 값`, `최대빈도 = 빈도`
    - 그렇지 않고 `빈도 = 최대빈도` **그리고** `값 < 최빈값` → `최빈값 = 값` (동률 시 작은 값 우선)
- 반복이 끝나면 `최빈값`을 말하기
- `흐름` 카테고리의 `N번 반복하기`를 **두 겹 중첩**, `판단` 카테고리의 `그리고`·`=`·`<` 블록을 활용하세요

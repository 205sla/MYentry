## 문제 설명

리스트 `숫자들`에서 **가장 자주 등장하는 값(최빈값)**을 찾아 말하기로 출력하세요. 동률이면 **더 작은 값**을 출력합니다.

### 입력
- `숫자들`: 1개 이상의 정수 리스트 (채점 시 자동 설정)

### 출력
- 최빈값을 말하기 (동률이면 더 작은 값)

### 예시
입력: `[1,2,2,3,3,3]` → 출력: `3`
입력: `[1,1,2,2]` → 출력: `1` (동률 → 작은 값)
입력: `[5]` → 출력: `5`

### 힌트
- `흐름` 카테고리의 반복 블록을 **두 겹 중첩**
- `판단`의 `그리고`·`=`·`<` 블록으로 동률 시 작은 값 선택
- 현재 최빈값·최대 빈도·순회 위치·현재 값의 빈도를 담을 보조 변수는 직접 만들어 쓰세요

<details>
<summary>자세한 접근법</summary>

- 현재 최빈값을 `숫자들의 1번째 항목`, 최대 빈도를 `0`으로 초기화
- 바깥 반복: 리스트를 훑으며 현재 항목을 "기준 값"으로 잡음
  - 안쪽 반복: 리스트를 다시 훑으며 기준 값과 같은 항목의 빈도 세기
  - 빈도 > 최대 빈도 → 최빈값과 최대 빈도 갱신
  - 빈도 = 최대 빈도 **그리고** 기준 값 < 최빈값 → 최빈값 갱신 (동률 시 작은 값 우선)
- 반복 끝나면 최빈값 말하기

</details>
